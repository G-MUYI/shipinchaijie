# Seedance 2.0（即梦）平台专属指南

当用户选择 Seedance 2.0 作为目标平台时，使用本指南生成专属格式的提示词。

## Seedance 平台特点

- 对自然流畅的中文描述理解能力很强
- 支持 @引用系统（@图片1、@视频1、@音频1）
- 提供十大能力模式
- 所有提示词和图片生成提示词必须使用中文

## 十大能力类型

### A) 纯文本生成
仅靠文字描述生成视频，不需要参考素材。

**提示词特点：**
- 详细的场景、角色、动作描述
- 明确的镜头运动和光线描述
- 情绪和氛围的具体表达

### B) 一致性控制
上传参考图保持角色/产品/场景统一。

**提示词格式：**
```
参考@图片1中的角色外观，保持完全一致。[场景描述]，[动作描述]，[镜头描述]。
```

**素材要求：**
- @图片1：角色/产品参考图（清晰、正面、光线均匀）

### C) 运镜/动作复刻
参考视频复刻镜头语言和动作。

**提示词格式：**
```
参考@视频1的运镜方式和动作节奏。[新场景描述]，[新角色描述]，保持原视频的镜头运动轨迹。
```

### D) 创意模板/特效复刻
模仿参考视频的转场和特效。

### E) 剧情创作/补全
根据图片或脚本自动生成剧情。

### F) 视频延长
对已有视频平滑延长。

**提示词格式：**
```
基于@视频1的结尾画面，继续延长[X]秒。[延长部分的动作描述]，保持画面连贯性和风格一致性。
```

### G) 声音控制
音色参考、对白、音效设计。

### H) 一镜到底
连贯长镜头，不切镜。

**提示词要求：**
- 描述完整的镜头运动轨迹（起点→过程→终点）
- 明确每个阶段的景别变化
- 强调画面的连贯性和流畅性

### I) 视频编辑
角色替换、剧情颠覆、元素增减。

**提示词格式：**
```
基于@视频1，将[原元素]替换为[新元素]，其他部分保持不变。
```

### J) 音乐卡点
画面节奏与音乐节拍匹配。

**提示词格式：**
```
参考@音频1的节奏，在[时间点]处卡点。[画面动作描述]与音乐节拍同步。
```

## 时长处理策略

### ≤15秒视频
使用时间戳分镜法，逐段描述。

**格式示例：**
```
0-3秒：[画面描述]，[运镜]，[动作]
3-7秒：[画面描述]，[运镜]，[动作]
7-12秒：[画面描述]，[运镜]，[动作]
12-15秒：[画面描述]，[运镜]，[动作]
```

### >15秒视频
使用分段拼接策略，将长视频拆分为多个15秒片段。

**拆分原则：**
- 每段独立完整，有明确的起承转合
- 段与段之间有清晰的转场设计
- 标注拼接点和转场方式

```markdown
## 使用指南

- 在即梦平台「全能参考」模式下使用
- 按素材清单准备并上传对应文件
- 在提示词输入框中输入"@"唤起素材引用
- 不要上传含写实真人脸部的素材
```

## 后续操作选项

生成基础提示词后，可以询问用户：

> Seedance 提示词已生成。你还可以：
>
> A) **视频延长** — 基于已生成的视频继续延长
> B) **视频编辑** — 对已有视频进行角色替换/元素修改
> C) **一镜到底优化** — 将分镜改为连续长镜头版本
> D) **换一种能力重新生成** — 使用其他 Seedance 能力
> E) **复刻重创** — 进入复刻流程
> F) **完成** — 满意了，结束
