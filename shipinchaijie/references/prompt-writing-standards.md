# 提示词写作标准

本文档定义了生成 AI 视频提示词时必须遵循的写作标准和质量要求。

## 手部描述渐进规则（极其重要！必须严格遵守）

人物手部的描述必须遵循**从自然到特效的渐进过程**。这是AI视频生成的关键规则，违反会导致生成失败。

### ❌ 常见错误示例

**错误1：开头就描述完整特效**
```
0-2秒：女巫伸出手，手上闪烁着紫色魔法光芒，指尖环绕着能量...
```
❌ 这样会让AI困惑：手一开始就有特效，还是后来才有？

**错误2：跳过触发过程**
```
0-2秒：手部自然状态
2-4秒：手部完全被火焰包裹
```
❌ 缺少中间的触发和蔓延过程。

### ✅ 正确的三阶段描述

### 1. 自然阶段（特效触发前）

**时间：** 视频开头的第1-2个时间段

只描述手部的自然外观：
- 皮肤颜色和质感
- 指甲颜色/形状/长度
- 手部姿态和动作
- 饰品（戒指、手镯等）

**正确示例：**
```
0-2秒：女巫的手（白皙肤色，修长手指，深紫色指甲）握着一根古老的木质法杖
```

**注意：** 此时特效可以在手持的物体上（如法杖发光），但手本身必须是自然的。

### 2. 触发瞬间（特效开始出现）

**时间：** 中间的2-3个时间段

精确描述特效如何从物体/触发点开始向手部蔓延：
- 触发动作（握紧、挥动、触碰等）
- 能量从哪里开始（法杖、魔法石、空气中）
- 如何蔓延到手部（顺着手臂、从指尖、从掌心）
- 蔓延的视觉效果（光线、纹路、颜色变化）

**正确示例：**
```
2-4秒：手掌猛然握紧法杖，法杖顶端爆发出强烈的紫色能量波
4-6秒：紫色能量顺着法杖向下流动，开始接触手掌，形成发光的魔法纹路
6-8秒：魔法纹路从手掌蔓延到手指，指甲开始泛起紫色荧光
```

这是"能力觉醒/激活"的关键瞬间，必须详细描述。

### 3. 特效阶段（特效完全激活后）

**时间：** 视频后半段的时间段

此时才描述手部带有完整的魔法/特效外观：
- 指甲变色/发光
- 能量流动效果
- 发光纹路/符文
- 特殊材质（金属化、晶体化、火焰化等）
- 周围的能量场/光晕

**正确示例：**
```
8-10秒：手部完全被紫色魔法能量包裹，指尖环绕着电弧状的能量流，
掌心浮现出发光的古老符文，整只手散发着强烈的紫色光晕
```

### 完整示例对比

**❌ 错误版本：**
```
0-2秒：女巫伸出手，手上闪烁着紫色魔法光芒
2-4秒：魔法能量更强了
4-6秒：手部被完全包裹
```

**✅ 正确版本：**
```
0-2秒：女巫的手（白皙肤色，修长手指，深紫色指甲）握着一根古老的木质法杖
2-4秒：手掌猛然握紧法杖，法杖顶端爆发出强烈的紫色能量波
4-6秒：紫色能量顺着法杖向下流动，开始接触手掌，形成发光的魔法纹路
6-8秒：魔法纹路从手掌蔓延到手指，指甲开始泛起紫色荧光
8-10秒：手部完全被紫色魔法能量包裹，指尖环绕着电弧状的能量流
```

### 为什么这个规则如此重要？

1. **AI生成逻辑：** AI视频生成工具需要理解"变化过程"，而不是"静态状态"
2. **视觉连贯性：** 渐进描述确保视频中的特效出现是自然的、有因果关系的
3. **避免混乱：** 如果开头就描述完整特效，AI会不知道是一直有特效，还是后来才出现

### 检查清单

在生成每个提示词后，检查：
- [ ] 第1-2个时间段：手部是否只描述了自然外观？
- [ ] 中间时间段：是否详细描述了触发动作和能量蔓延过程？
- [ ] 后半段时间段：是否才开始描述完整的特效状态？
- [ ] 整体：是否能清晰看出"自然→触发→完整特效"的三阶段过渡？

## 总述区写作要求

### 核心主题
一句话概括视频核心概念、风格定位和情绪基调。

**好的示例：**
```
一位神秘女性在雨夜台北街头的自拍式奔跑记录，
手持摄影的真实感与浪漫氛围交织
```

**差的示例：**
```
一个视频
```

### 角色设定
完整的外观描述，包含：
- 面部特征（五官、肤色、表情基调）
- 发型（长度、颜色、造型）
- 服装（款式、颜色、材质、细节）
- 体态（身高感、体型、姿态）

如有参考图，标注"严格1:1复刻参考图"。

### 视觉风格与色调
必须包含以下所有元素：
1. **摄影机/镜头参考** — 如"IMAX电影质感，Panavision C系列35mm镜头，f/4光圈"
2. **主色调方案** — 主色+辅色+点缀色
3. **阴影处理** — 阴影的深度、边缘、色彩倾向
4. **质感特征** — 颗粒感、清晰度、对比度
5. **视觉对比主题** — 如"温暖人物 vs 冷色环境"

### 光线与质感
1. **光线类型和方向** — 自然光/人工光/混合光，来自哪个方向
2. **材质渲染要求** — SSS（次表面散射）、布料纤维、金属反射等
3. **整体渲染风格** — 写实/风格化/超现实
4. **真实感渲染技术** — 全局光照、次表面散射、体积光、HDR

### 镜头运动
用箭头描述完整运镜路径：

**示例：**
```
极限广角坠落 → 晃动推近至中景特写 → 变形环绕 → 拉远低角度双人构图
```

不要只写"手持跟拍"这种笼统描述。

## 时间线区写作要求

### 段落主题标签
每段开头用简洁标签概括该段核心。

**示例：**
- "凝视"
- "激活"
- "撕裂"
- "重生"

### 画面描述要具象
不写抽象概念，写具体可见的画面元素。

**好的示例：**
```
台北街头骑楼下，雨帘形成模糊光斑，
远处霓虹灯的红蓝色反射在湿润的地面上
```

**差的示例：**
```
美丽的场景
```

### 动作要精确
精确到肢体细节，不用笼统动词。

**好的示例：**
```
她一只手举机对着自己，另一只手拉着某人奔跑，
脚步踩在积水上溅起水花
```

**差的示例：**
```
她在走路
```

### 情绪要具体
不写抽象情绪词，写具体的表情和肢体语言。

**好的示例：**
```
抬头对镜头露出无奈又好笑的表情，
眉毛微挑，嘴角上扬，眼神中带着调侃
```

**差的示例：**
```
她很开心
```

### 光线与色调逐段描述
每段标注该时刻的光线变化和色彩倾向。

**示例：**
```
逆光剪影，暗金色能量照亮面部，
形成强烈的明暗对比
```

### 运镜描述轨迹
不只写运镜类型，要描述具体的运动轨迹。

**好的示例：**
```
镜头跟随坠落、落地震动，快速推近面部特写，
从广角切换到中焦段
```

**差的示例：**
```
手持跟拍
```

### 特效分段细化
每段描述该时刻的特效细节，不要笼统概括。

**示例：**
```
暗金属碎片向外迸发，装甲以吞噬方式组装，
从手臂开始向肩部蔓延
```

### 台词带语气
台词必须标注说话的语气和情绪。

**格式：**
```
台词（喘着气又笑）："真是的，又淋成落汤鸡了"
```

### 无台词段落标注声音
没有台词的段落也要描述声音设计。

**格式：**
```
[雨声+街道环境音+远处车流声，无台词]
```

## 真实感渲染描述（极重要）

总述区和每段必须包含真实感渲染关键词：
- 次表面散射（SSS）
- 全局光照（GI）
- 体积光（Volumetric Lighting）
- 物理正确的布料褶皱
- 微表面细节
- 环境光遮蔽（AO）
- HDR 光照
- 景深效果（DOF）

## 分段粒度要求

- **基本单位：** 1-2秒一段
- **关键动作：** 精确到0.5秒
- **高潮段落：** 可以更细，0.3秒一个关键帧

## 动作连贯性与物理逻辑（极其重要！）

AI视频生成的核心挑战之一是确保动作的连贯性和符合物理规律。以下规则必须严格遵守。

### 1. 动作分解原则

**每个动作必须分解为完整的动作链：**

❌ **错误示例：**
```
0-2秒：角色站立
2-4秒：角色跳跃到空中
4-6秒：角色落地
```
问题：缺少起跳准备、空中姿态变化、落地缓冲等中间过程。

✅ **正确示例：**
```
0-1秒：角色站立，双脚分开与肩同宽，身体重心稳定
1-2秒：双膝微曲，身体重心下沉，双臂向后摆动（起跳准备）
2-3秒：双腿用力蹬地，身体向上跃起，双臂向上摆动辅助起跳
3-4秒：身体在空中达到最高点，双臂自然展开保持平衡，双腿微曲
4-5秒：身体开始下降，双腿准备着地，双臂向下摆动
5-6秒：双脚接触地面，膝盖弯曲缓冲冲击力，身体前倾保持平衡
6-7秒：身体重心恢复稳定，双腿伸直，回到站立姿态
```

**动作链完整性检查清单：**
- [ ] 准备动作（蓄力、重心转移、肌肉紧绷）
- [ ] 执行动作（主要动作的完整过程）
- [ ] 惯性延续（动作完成后的余势）
- [ ] 恢复动作（回到稳定状态）

### 2. 物理规律遵守

**重力与惯性：**
- 跳跃必须有上升→最高点→下降的完整弧线
- 奔跑时身体前倾，停止时身体后仰（惯性）
- 物体抛出后遵循抛物线轨迹
- 落地时必须有缓冲动作（膝盖弯曲、身体下沉）

**力的传递：**
- 拳击：力量从腿部→腰部→肩部→手臂→拳头
- 挥剑：力量从腰部旋转→肩部带动→手臂挥动→剑刃划过
- 投掷：重心后移→转体蓄力→手臂挥动→物体脱手

**动量守恒：**
- 快速移动后突然停止，身体会有惯性前倾
- 旋转动作需要反向扭腰来启动和停止
- 重物落地会产生地面震动和尘土飞扬

✅ **正确示例（拳击动作）：**
```
0-1秒：角色站立，左脚在前，右脚在后，身体侧对目标，双拳护在面前
1-2秒：右腿蹬地，力量传递到腰部，腰部开始向左旋转
2-3秒：腰部旋转带动右肩向前，右臂开始伸展，左拳回收护头
3-3.5秒：右拳全力向前击出，手臂完全伸直，拳头击中目标
3.5-4秒：拳头接触目标瞬间，目标向后震动，空气产生冲击波纹
4-5秒：右拳快速回收，身体重心恢复，左脚向前一步稳定身体
```

### 3. 战斗动作编排规则

**攻击动作三要素：**
1. **蓄力阶段** — 身体准备，能量聚集，肌肉紧绷
2. **爆发阶段** — 力量释放，动作执行，特效触发
3. **收招阶段** — 动作完成，身体恢复，准备下一个动作

**防御与闪避：**
- 格挡：手臂抬起→武器/盾牌接触攻击→身体后退卸力→恢复姿态
- 闪避：观察→身体侧移/后仰→攻击擦身而过→恢复平衡

**连招设计：**
- 每个动作的结束姿态必须是下一个动作的起始姿态
- 连招之间用小幅度的重心转移和身体调整连接
- 避免"瞬移"式的姿态突变

✅ **正确示例（三连击）：**
```
0-1秒：左拳向前直刺，身体重心前移
1-2秒：左拳回收的同时，右拳从腰间发力向前勾拳，身体微微右转
2-3秒：右拳击中后，借助右转的惯性，左腿抬起向前踢出
3-4秒：左腿落地，身体重心稳定，双拳回到防御姿态
```

### 4. 运镜与动作的配合

**镜头运动必须服务于动作：**

**跟随运镜：**
- 角色奔跑时，镜头以相同速度跟随，保持角色在画面中心
- 角色跳跃时，镜头同步上升和下降
- 角色转身时，镜头环绕或跟随旋转

**强调运镜：**
- 重击瞬间：镜头快速推近+轻微震动
- 闪避瞬间：镜头跟随身体快速侧移，产生动态模糊
- 蓄力瞬间：镜头缓慢推近，营造紧张感

**第一人称POV特殊规则：**
- 奔跑时镜头上下轻微晃动（模拟步伐节奏）
- 转头时镜头水平旋转，边缘产生动态模糊
- 受击时镜头剧烈晃动+短暂失焦
- 手臂动作时，手臂从画面下方或侧方伸入画面

**第三人称特殊规则：**
- 展示全身动作时使用全景或中景
- 关键动作瞬间切换到特写（如拳头击中、剑刃划过）
- 使用环绕镜头展示360度动作（如旋转攻击）

### 5. 时间感与节奏控制

**慢动作使用规则：**
- 仅在关键瞬间使用（击中瞬间、特效爆发、情绪高潮）
- 慢动作前后必须有正常速度的过渡
- 慢动作时增加细节描述（汗水飞溅、衣物飘动、粒子轨迹）

**快节奏动作：**
- 使用0.5秒甚至0.3秒的细分段
- 每段描述一个明确的动作关键帧
- 用"快速"、"瞬间"、"闪电般"等词强调速度

**节奏变化：**
- 战斗前：慢节奏，蓄势待发（1-2秒一段）
- 战斗中：快节奏，连续动作（0.5秒一段）
- 战斗后：慢节奏，恢复平静（1-2秒一段）

### 6. 环境交互与反馈

**角色与环境的物理交互：**
- 脚步踩在不同地面产生不同效果（水面溅起水花、沙地扬起尘土、雪地留下脚印）
- 身体碰撞墙壁/物体时，物体产生相应反应（墙壁震动、物体倒塌、尘土飞扬）
- 风吹动衣物、头发、周围的树叶/窗帘
- 光线随角色移动产生阴影变化

**特效与环境的交互：**
- 火焰照亮周围环境，产生动态光影
- 冰霜特效使周围空气产生雾气
- 雷电特效照亮整个场景，产生闪光
- 能量波扩散时推动周围的尘土/水面/物体

✅ **正确示例：**
```
2-3秒：角色右拳击出，拳头周围包裹着火焰能量
3-3.5秒：火焰拳击中地面，地面瞬间龟裂，裂缝中涌出火光
3.5-4秒：冲击波向外扩散，推动周围的尘土和碎石向外飞溅
4-5秒：火焰在地面燃烧，照亮角色的面部和周围环境，产生跳动的光影
```

### 7. 连贯性检查清单

生成每个提示词后，必须检查：

**动作连贯性：**
- [ ] 每个动作是否有完整的准备→执行→恢复过程？
- [ ] 相邻时间段的动作是否自然衔接？
- [ ] 是否有"瞬移"式的姿态突变？
- [ ] 连续动作的重心转移是否合理？

**物理逻辑：**
- [ ] 跳跃/落地是否符合重力规律？
- [ ] 快速移动/停止是否考虑惯性？
- [ ] 力量传递路径是否正确（腿→腰→肩→手）？
- [ ] 物体运动轨迹是否符合物理规律？

**战斗动作：**
- [ ] 攻击是否有蓄力→爆发→收招三阶段？
- [ ] 连招之间是否有合理的衔接动作？
- [ ] 防御/闪避动作是否完整？
- [ ] 受击反应是否符合力的方向？

**运镜配合：**
- [ ] 镜头运动是否服务于动作展示？
- [ ] 关键动作是否有镜头强调？
- [ ] 第一人称/第三人称视角是否正确应用？
- [ ] 镜头切换是否自然？

**环境交互：**
- [ ] 角色与地面/物体的交互是否描述？
- [ ] 特效对环境的影响是否体现？
- [ ] 光影变化是否随动作变化？
- [ ] 声音设计是否与动作匹配？

### 8. 常见错误与修正

**错误1：动作跳跃**
```
❌ 0-2秒：角色站立
   2-4秒：角色已经在空中飞行
```
```
✅ 0-1秒：角色站立，双腿微曲准备起跳
   1-2秒：双腿用力蹬地，身体开始离地
   2-3秒：身体在空中上升，双臂展开
   3-4秒：身体在空中飞行，保持平衡姿态
```

**错误2：忽略惯性**

## 多主题视频拆分规则（关键！）

### 什么是多主题视频？

满足以下任一条件即为多主题视频：
1. **不同角色：** 视频中有多个主角，每个主角有独立的故事线
2. **不同场景：** 视频包含多个完全不同的场景（如室内→室外→太空）
3. **不同风格：** 视频的不同段落有完全不同的视觉风格（如现实→梦境→回忆）
4. **用户明确分段：** 用户说"0-10秒是...，10-20秒是...，20-30秒是..."

### 多主题视频的处理方式

**必须为每个主题生成独立的完整提示词文件！**

### ❌ 错误做法

生成一个文件，里面包含所有主题：
```markdown
# 三系法师视频提示词

## 总述区
三个不同的法师...

## 时间线区
0-10秒：火系法师...
10-20秒：水系法师...
20-30秒：雷系法师...
```

❌ 问题：
- 用户无法单独生成某一段
- 角色描述混乱（三个法师的外观写在一起）
- 无法针对单个主题调整参数

### ✅ 正确做法

### 拆分规则详解

1. **独立的总述** — 每个提示词有自己的：
   - 时长（通常每个主题10-15秒）
   - 画幅比例
   - 核心主题描述
   - 角色设定（外观、服装、配饰）
   - 视觉风格与色调（与该主题匹配）
   - 光线与质感
   - 镜头运动

2. **主题匹配的细节** — 角色外观必须与该段落主题完全匹配：
   - 火系段落 → 岩浆纹路指甲 + 火焰纹护手 + 红金色袖口
   - 水系段落 → 水波流动甲片 + 海蓝宝石指环 + 深蓝薄纱
   - 雷系段落 → 紫色电弧指甲 + 雷电纹手套 + 紫白色袖口

3. **统一时长** — 每个独立提示词的时长统一（默认10-15秒）

4. **内部分段** — 每个提示词内部按1-2秒分段，确保至少5-7个时间段

5. **文件命名清晰** — 使用主题名称标注：
   - `video-breakdown-{主题名}-{timestamp}.md`
   - 例如：`video-breakdown-火系法师-20260329.md`

### 为什么要拆分？

1. **便于生成：** 用户可以逐个生成后拼接，也可以只生成其中某一段
2. **便于调整：** 如果某一段不满意，只需重新生成该段，不影响其他段
3. **参数独立：** 每个主题可以使用不同的生成参数（如种子值、风格强度）
4. **角色清晰：** 每个文件只描述一个角色，避免混淆

- [ ] 是否为每个主题生成了独立的文件？
- [ ] 每个文件是否有完整的总述区和时间线区？
- [ ] 每个文件的角色描述是否与该主题完全匹配？
- [ ] 文件命名是否清晰标注了主题？
- [ ] 是否告诉用户生成了几个文件，每个对应哪个主题？
