# 十二星座视频提示词生成规则

## 核心调整（2026-03-27）

### 1. 每个星座独立提示词

**问题**：之前没有按照每个星座单独生成提示词，没有明显区分

**解决方案**：
- 每个星座生成完全独立的提示词文档
- 每个提示词开头标注星座名称（如"# 白羊座专属魔法球"）
- 不同星座之间用明显分隔符区分
- Gemini分析时自动识别视频对应的星座

### 2. 首帧AI绘画提示词

**问题**：每个星座开始没有给出描述首帧的AI绘画提示词

**解决方案**：
每个星座提示词必须包含首帧AI绘画提示词，格式如下：

```markdown
# [星座名称]专属魔法球

## 首帧AI绘画提示词

电影级写实摄影，真实拍摄质感。[手部状态详细描述：皮肤质感、指甲细节、手部姿态]。手中悬浮/握持一颗乒乓球大小的[颜色]魔法球，[魔法球材质、光泽、透明度描述]。[场景环境描述：背景、光线、氛围]。次表面散射的真实皮肤，物理正确的光影，全局光照，体积光穿透空气，HDR高动态范围。照片级写实，电影实拍质感，避免3D渲染感。
```

**必须包含的元素**：
- 手部自然状态（皮肤、指甲、姿态、饰品）
- 魔法球描述（大小、颜色、材质、光泽）
- 场景环境（背景、光线、氛围）
- 真实质感关键词

### 3. 避免CG动画关键词

**问题**：提示词中出现"CG动画"等关键词，导致画面不够真实

**解决方案**：

**严禁使用的词汇**：
- CG动画
- 3D渲染
- 动画风格
- 卡通效果
- 塑料质感

**必须使用的词汇**：
- 电影实拍质感
- 真实摄影机拍摄
- 照片级写实
- 电影级写实摄影
- 次表面散射的真实皮肤
- 物理正确的布料褶皱
- 真实金属菲涅尔反射
- 全局光照柔和弹射
- 体积光穿透空气微粒
- HDR高动态范围

## 使用流程

1. 分析视频（自动识别星座）：
```bash
python bin/gemini-analyze.py video.mp4 --output analysis.json
```

2. 生成独立提示词：
```bash
python bin/generate-prompts.py analysis.json --output prompts/白羊座.md
```

3. 每个星座的提示词包含：
   - 星座标识
   - 首帧AI绘画提示词
   - 完整视频生成提示词
   - 时间线描述
