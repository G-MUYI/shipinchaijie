# Seedance 2.0（即梦）平台专属指南

当用户选择 Seedance 2.0 作为目标平台时，使用本指南生成专属格式的提示词。

## Seedance 平台特点

- 对自然流畅的中文描述理解能力很强
- 支持 @引用系统（@图片1、@视频1、@音频1）
- 提供十大能力模式
- 所有提示词和图片生成提示词必须使用中文

## Seedance 提示词核心原则（极重要！）

以下原则是 Seedance 2.0 区别于通用平台（Sora/可灵/Runway）的关键差异，违反任何一条都会显著降低生成质量。

### 原则1：提示词长度控制（灵活范围）

基于 1319 条真实数据分析，Seedance 2.0 提示词长度分布：
- **简单场景（<100字）**：9%，适合单一动作/趣味短视频
- **标准长度（100-500字）**：50%，大多数场景的最佳范围
- **详细描述（500-800字）**：15%，复杂叙事/多场景
- **超长提示（>800字）**：23%，完整故事/多主题

**推荐策略：**
- 简单场景：100-300字
- 标准场景：300-600字（中位数 336 字）
- 复杂场景：600-1000字
- 不要盲目压缩，根据内容复杂度调整

**精简策略：**
1. 砍掉冗余修饰词，只保留视觉动作和物体特征
2. 每段只保留核心字段：`动作 + 镜头 + 画面`，不需要每段都写全十个字段
3. 用 `@` 素材代替文字描述（角色外观用 `@图片` 比文字稳定10倍）
4. 光影和氛围在开头总述一次，时间线区不重复

**❌ 错误：过长的 Seedance 提示词（420字）**
```
电影级写实摄影，真实拍摄质感。第一人称POV视角，一只白皙纤细的女性右手，淡粉色椭圆形指甲，手心向上轻轻托举一颗乒乓球大小的深蓝色透明魔法球，球体内部有神秘符文发光。背景虚化为昏暗的地下神殿，石柱上刻满发光的符文，微弱的蓝色光芒从地面裂缝中透出，阴沉氛围。光线：主光源来自地面裂缝的蓝色冷光，从下往上照亮手部和魔法球，营造神秘氛围。辅助光源是符文的微弱荧光，点缀在背景中。全局光照，次表面散射皮肤材质，体积光穿透空气微粒。0-3秒：手掌向上摊开，魔法球悬浮旋转，球体内部符文发光闪烁。运镜：缓慢推近手部特写。光线：蓝色冷光从下方照亮。3-5秒：手指收拢捏爆魔法球，碎裂闪光，能量向手臂蔓延。运镜：镜头震动。光线：白光闪烁后恢复蓝色基调。5-8秒...
```

**✅ 正确：精简的 Seedance 提示词（180字）**
```
@图片1为首帧。第一人称POV视角，昏暗地下神殿，蓝色冷光从地面裂缝透出。

0-3秒：缓慢推近手部特写，右手托举乒乓球大小的蓝色魔法球，球内符文旋转发光。
3-5秒：镜头震动，五指收紧捏爆魔法球，碎裂闪光，蓝色能量沿手指向手臂蔓延。
5-8秒：镜头前推，双手推出蓝色能量波，击中前方暗影生物，化为黑烟消散。

电影级写实摄影，ARRI Alexa Mini LF，画面稳定，4K。
禁止出现文字、字幕、LOGO或水印。
```

### 原则2：单运镜原则

**Seedance 2.0 在单条提示词中混写多种运镜极易导致画面崩坏。** 每个时间段应该只描述一种运镜方式。

**与通用平台的对比：**
- 通用平台：可以在提示词中使用运镜轨迹箭头法（`推近 → 跟随 → 环绕 → 拉远`）
- Seedance 2.0：每段只写一种运镜，不在一句话中连续切换

**❌ 错误：一段中混写多种运镜**
```
0-5秒：镜头从远景快速推进到特写，然后环绕主角旋转180度，最后拉远到全景。
```

**✅ 正确：每段只写一种运镜**
```
0-2秒：远景固定镜头，展示主角站在街道中央。
2-4秒：缓慢推进到面部特写，聚焦表情变化。
4-6秒：镜头环绕主角旋转，展示周围环境。
```

**如果需要复杂运镜：**
- 方案A：分段生成，每段一种运镜，后期拼接
- 方案B：上传运镜参考视频 `@视频1`，用 `"参考@视频1的运镜方式"` 代替文字描述

### 原则3：禁止文学化/隐喻描述

**Seedance 2.0 对文学隐喻、诗意表达、抽象概念的理解力极弱。** 必须直接描述可见的视觉画面。这是 Seedance 效果差的头号原因。

**❌ 错误示例与 ✅ 修正：**

| ❌ 文学化描述（效果差） | ✅ 直白视觉描述（效果好） |
|----------------------|------------------------|
| 举头望明月，思乡之情油然而生 | 一位古代男子，抬头凝视天空中的圆月，神情忧郁，眼角微湿 |
| 时光在指尖流淌 | 女性手指翻动旧照片，照片泛黄卷边 |
| 春风化雨，万物复苏 | 细雨落在嫩绿的树叶上，水珠沿叶脉滑落，花苞缓慢绽开 |
| 生命的律动在空气中跳跃 | 一只蝴蝶在花丛中扇动翅膀，飞向阳光方向 |
| 夜色温柔地拥抱了城市 | 俯瞰城市夜景，街道路灯逐渐亮起，建筑窗户透出暖黄色光 |
| 心碎的声音在回响 | 女性低头，一滴眼泪从脸颊滑落，滴在手中的照片上 |

**规则：**
1. 每句话必须描述一个**可以被摄影机拍到的画面**
2. 如果你写的内容无法用相机拍下来，就需要改写
3. 情绪通过具体的**表情、肢体、动作**来传达，不用抽象情绪词
4. 环境通过具体的**物体、光线、颜色**来传达，不用氛围形容词

### 原则4：素材（@引用）与文字的分工策略

在 Seedance 2.0 中，**素材的控制权重大于文字权重**。如果 `@` 素材和文字描述冲突，模型优先跟随素材。因此，拆解视频时需要额外判断"哪些元素用素材控制，哪些用文字描述"。

**素材 vs 文字分工决策表：**

| 控制目标 | 推荐方式 | 原因 |
|---------|---------|------|
| 角色外观/一致性 | `@图片` 锁脸 | 比文字描述稳定10倍，防止角色漂移 |
| 运镜轨迹/节奏 | `@视频` 复刻 | 文字描述复杂运镜容易崩，视频参考更精准 |
| 音乐卡点/节拍 | `@音频` 同步 | 文字无法精确控制每个节拍的时间点 |
| 首帧画面 | `@图片` 首帧 | 锁定视频起始画面，防止"抽卡"式随机开头 |
| 场景切换关键帧 | `@图片` 多张 | 每个场景一张关键帧图，防止场景漂移 |
| 动作细节/肢体 | **文字描述** | 素材无法精确指定"哪只手做什么动作" |
| 光影氛围/色调 | **文字描述** | 更灵活，可逐段调整冷暖变化 |
| 特效类型/过程 | **文字描述** | 特效的渐进变化需要文字逐段控制 |
| 情绪/表情 | **文字描述** | 微表情变化需要文字精确指导 |

**实际应用示例：**

拆解一个"女性魔法师在森林中施法"的视频时：
- 角色外观 → 生成 `@图片1` 的绘图提示词，上传为参考图
- 森林场景 → 生成 `@图片2` 的场景参考图
- 运镜轨迹 → 如有参考视频，上传为 `@视频1`
- 施法动作、特效过程、表情变化 → 用文字在提示词中逐段描述

**素材优先级排序（当素材数量有限时）：**
1. 角色锁脸图（最高优先级，解决最大痛点）
2. 首帧画面图（确定视频起点）
3. 运镜参考视频（复杂运镜时必备）
4. 场景关键帧图（多场景切换时需要）
5. 音频参考（音乐卡点时需要）

## 十大能力类型

### A) 纯文本生成
仅靠文字描述生成视频，不需要参考素材。

**提示词特点：**
- 详细的场景、角色、动作描述
- 明确的镜头运动和光线描述
- 情绪和氛围的具体表达

**实际案例：魔法师战斗场景**
```
一位身穿深蓝色法袍的年轻女性魔法师，站在昏暗的地下神殿中央。她伸出右手，手心向上托举着一颗乒乓球大小的深蓝色魔法球，球体内部有神秘符文发光。镜头从中景缓慢推进到手部特写，背景的石柱上刻满发光的符文。

突然，她五指收紧，魔法球在掌心碎裂，蓝色能量从爆裂点向外扩散，沿着手指蔓延到整个手臂。镜头快速推进，捕捉能量蔓延的细节。

她双手向前推出，释放出强大的蓝色能量波，冲向前方的暗影生物。镜头跟随能量波快速前推，暗影生物被击中后化为黑色烟雾消散。

光线：主光源来自地面裂缝的蓝色冷光，辅助光源是符文的微弱荧光。电影级写实摄影，ARRI Alexa Mini LF拍摄，Sigma 35mm f/1.4镜头。
```

**最佳实践：**
- 使用具体的数字描述（"乒乓球大小"而非"小球"）
- 明确光源方向和类型
- 包含摄影机和镜头参考
- 描述动作的完整过程（准备→执行→结束）

### B) 一致性控制
上传参考图保持角色/产品/场景统一。

**提示词格式：**
```
参考@图片1中的角色外观，保持完全一致。[场景描述]，[动作描述]，[镜头描述]。
```

**素材要求：**
- @图片1：角色/产品参考图（清晰、正面、光线均匀）

**实际案例：角色一致性保持**
```
参考@图片1中的女性魔法师外观，保持完全一致（深蓝色法袍、银色长发、紫色眼睛、手持木质法杖）。

她站在森林空地中央，阳光从树叶缝隙洒下，形成斑驳光影。她举起法杖，杖顶的水晶开始发光，周围的树叶被魔法力量吹起，在空中旋转。镜头从全景缓慢推进到中景，展示她专注的表情和飘动的法袍。

光线：自然光从上方照射，形成柔和的顶光效果。电影级写实摄影，保持与参考图相同的光线质感和色调风格。
```

**@图片1生成提示词：**
```
电影级写实摄影，真实拍摄质感。一位年轻女性魔法师，深蓝色长款法袍，银色长发自然垂落，紫色眼睛，手持古老的木质法杖。正面站立姿态，表情专注而神秘。背景虚化为森林环境，柔和的自然光从上方照射。

ARRI Alexa Mini LF拍摄，Sigma 50mm f/1.8镜头，光圈f/2.8，自然光照明，色彩分级偏冷调，照片级写实。避免3D渲染感、CG动画风格、卡通化效果。
```

**最佳实践：**
- 参考图必须清晰展示角色的关键特征（发型、服装、配饰）
- 在提示词中明确列出需要保持一致的特征
- 光线风格应与参考图保持一致
- 避免要求角色做参考图中没有展示的复杂动作

### C) 运镜/动作复刻
参考视频复刻镜头语言和动作。

**提示词格式：**
```
参考@视频1的运镜方式和动作节奏。[新场景描述]，[新角色描述]，保持原视频的镜头运动轨迹。
```

**实际案例：运镜复刻**
```
参考@视频1的运镜方式和动作节奏（手持跟拍→快速推进→环绕旋转→拉远定格）。

场景替换为现代都市夜景，霓虹灯闪烁的街道。角色替换为一位身穿黑色风衣的年轻男性，短发，手持智能手机。

0-3秒：手持跟拍，镜头跟随角色在雨夜街道奔跑，路面反射着霓虹灯光。
3-6秒：镜头快速推进到手机特写，屏幕上显示神秘的蓝色光芒。
6-9秒：镜头环绕角色旋转180度，展示周围的城市夜景和雨滴。
9-12秒：镜头拉远到全景，角色站在街道中央，周围是模糊的车流光轨。

保持原视频的镜头运动速度、景别切换节奏和转场方式。电影级写实摄影，手持摄影机的真实晃动感。
```

**最佳实践：**
- 明确列出原视频的运镜轨迹（用箭头连接）
- 保持相同的时间分段和节奏
- 新场景和角色要与运镜风格匹配
- 标注关键的镜头运动参数（速度、角度、景别）

### D) 创意模板/特效复刻
模仿参考视频的转场和特效风格。

**提示词格式：**
```
参考@视频1的特效风格和转场方式。[新场景描述]，使用相同的特效类型和转场节奏。
```

**素材要求：**
- @视频1：特效/转场参考视频（清晰展示特效细节和转场方式）

**实际案例：粒子特效复刻**
```
参考@视频1的粒子消散特效和闪白转场方式。

一位街头魔术师站在夜晚的城市广场，手中握着一张扑克牌（红心A）。他轻轻弹指，扑克牌边缘开始发光，然后整张牌分解成红色和金色的细小粒子。

粒子形成一条流动的光带，螺旋上升飞向远处的大楼玻璃幕墙。粒子撞击玻璃瞬间，整个幕墙亮起，形成巨大的红心A图案。

闪白转场（参考@视频1的转场节奏）。

魔术师伸手向空中一抓，粒子从大楼反向飞回，在他手中重新凝聚成完整的扑克牌。

电影级写实摄影，粒子特效具有真实的光照衰减和运动轨迹。
```

**@视频1要求：**
- 清晰展示特效的触发、运动、消散过程
- 转场方式明确（闪白/溶解/粒子过渡等）
- 特效与环境的交互清晰可见

**最佳实践：**
- 明确列出需要复刻的特效类型（粒子/光效/扭曲/爆炸等）
- 描述特效的运动轨迹和速度
- 标注转场的时间点和方式
- 保持特效的物理真实感（光照、衰减、碰撞）

### E) 剧情创作/补全
根据图片或脚本自动生成剧情，补全故事的前因后果。

**提示词格式：**
```
基于@图片1的场景，生成[前置/后续]剧情。[剧情描述]，保持角色和场景风格一致。
```

**素材要求：**
- @图片1：关键场景图（清晰展示角色、环境、情绪状态）

**实际案例：婚礼场景剧情补全**
```
基于@图片1（新郎新娘在教堂门口的场景），生成婚礼前的准备剧情。

0-3秒：新娘在化妆间对着镜子整理婚纱，母亲在身后帮她戴上头纱。阳光从窗户洒进来，照亮白色婚纱的蕾丝细节。新娘深呼吸，眼神中有紧张也有期待。

3-6秒：新郎在教堂侧门整理领结，伴郎拍拍他的肩膀。新郎看向教堂大门方向，手指轻轻摩挲着戒指盒。

6-9秒：教堂大门缓缓打开，新娘挽着父亲的手臂出现在门口。阳光从背后照射，形成逆光剪影。新郎转身看到新娘，表情从紧张变为惊喜和感动。

9-12秒：镜头切换到@图片1的场景，新郎新娘站在教堂门口，四目相对，周围宾客鼓掌。

保持@图片1中的角色外观、服装风格和光线氛围。电影级写实摄影，自然光照明。
```

**最佳实践：**
- 明确补全的是前置剧情还是后续剧情
- 保持与参考图的风格、色调、情绪一致
- 描述清晰的时间线和因果关系
- 包含角色的情绪变化和动作细节

### F) 视频延长
对已有视频平滑延长，保持动作和风格的连贯性。

**提示词格式：**
```
基于@视频1的结尾画面，继续延长[X]秒。[延长部分的动作描述]，保持画面连贯性和风格一致性。
```

**素材要求：**
- @视频1：需要延长的原视频（结尾画面清晰，动作未完成或可延续）

**实际案例：舞蹈动作延长**
```
基于@视频1的结尾画面（舞者完成旋转，右脚刚落地），继续延长5秒。

舞者右脚落地后，身体重心前移，左腿向后抬起形成阿拉贝斯克姿态。双臂从身体两侧缓慢展开，指尖延伸。

镜头缓慢环绕舞者旋转，展示完整的姿态。背景霓虹灯光继续闪烁，与@视频1的节奏保持一致。

舞者保持姿态2秒后，左腿缓慢放下，身体转向镜头，做出结束姿势。

保持@视频1的光线风格（霓虹冷暖对比）、运镜速度和音乐节奏。电影级写实摄影。
```

**注意事项：**
- 延长部分的第一帧必须与原视频最后一帧无缝衔接
- 保持相同的光线、色调、运镜风格
- 动作必须符合物理逻辑，不能突变
- 如有音乐，延长部分需与节奏匹配

**最佳实践：**
- 仔细观察原视频结尾的动作状态（姿态、速度、方向）
- 描述延长部分的完整动作链（不要跳跃）
- 明确延长的时长（建议3-5秒）
- 标注需要保持一致的元素（光线/色调/运镜/音乐）

### G) 声音控制
音色参考、对白、音效设计的精确控制。

**提示词格式：**
```
参考@音频1的音色和节奏。[场景描述]，[对白内容]，[音效设计]。
```

**素材要求：**
- @音频1：音色/音乐参考（如有）

**实际案例：对白场景**
```
高级餐厅烛光晚餐场景，暖黄色调，柔和爵士乐背景。

0-2秒：中景，丈夫推过丝绒礼盒，妻子抬眼看向他。
对白（丈夫，低沉温柔）："亲爱的，今天辛苦你了。"
音效：刀叉轻碰瓷盘声、柔和爵士BGM。

2-4秒：特写礼盒打开，定制耳机在烛光下泛光。妻子倒吸一口气。
对白（妻子，哽咽）："这是……？"
对白（丈夫）："你总说加班戴耳机吵，这个，只听得到我的声音。"
音效：礼盒开合轻响、妻子气音、BGM渐弱。

4-6秒：近景妻子落泪，丈夫伸手拭泪。
对白（妻子，哭着笑）："你怎么知道我想要这个很久了……"
音效：细微抽噎声、温柔钢琴音加入。

6-8秒：特写丈夫为妻子戴上耳机，两人交握的手。
对白（丈夫，轻声）："以后，世界再吵，我都在你耳边。"
音效：BGM达到高潮后渐弱，只剩呼吸声。

电影级写实摄影，自然光照明。
```

**音效设计三层结构**：
- 音乐层：爵士乐、钢琴音
- 环境音层：餐厅环境音、刀叉碰撞声
- 动作音层：礼盒开合声、呼吸声、抽噎声

**最佳实践：**
- 对白标注语气和情绪（低沉温柔/哽咽/轻声）
- 音效按时间段分层描述
- BGM的强弱变化与情绪同步
- 关键时刻使用"只剩XX声"强调氛围

### H) 一镜到底
连贯长镜头，不切镜。

**提示词要求：**
- 描述完整的镜头运动轨迹（起点→过程→终点）
- 明确每个阶段的景别变化
- 强调画面的连贯性和流畅性
- **场景切换时必须标注物理遮挡物**（利用柱子、墙壁、烟雾、光效、快速横移模糊等作为遮挡，实现无缝转场）

**实际案例：战斗场景一镜到底**
```
15秒一镜到底战斗场景，第一人称POV视角。

0-3秒：镜头从低角度仰视开始，三只暗影生物从前方冲来。镜头轻微上下晃动（奔跑节奏），右手伸入画面下方，手中托举着乒乓球大小的火红色魔法球。

3-5秒：镜头快速向右侧移（闪避动作），产生动态模糊。同时右手在画面中央做出握拳手势，魔法球在掌心碎裂，火焰能量从爆裂点向手部蔓延。

5-8秒：镜头稳定，焦点在手部。火焰能量完全包裹右手，手臂向前推出，释放火焰冲击波。镜头跟随火焰快速前推，暗影生物被击中后化为黑烟。

8-11秒：镜头快速向上仰视，Boss从天而降，落地产生冲击波。镜头剧烈震动，短暂失焦。

11-15秒：镜头恢复稳定，双手在画面中央结印，火焰能量在双手间汇聚成巨大火球。镜头缓慢推进，营造紧张感。火球释放瞬间，镜头快速前推并闪白。

全程保持第一人称视角，镜头运动模拟头部和身体的自然移动。电影级写实摄影，手持摄影机质感。
```

**最佳实践：**
- 明确标注每个时间段的镜头运动方式
- 描述镜头运动的物理原因（奔跑、闪避、受击）
- 第一人称POV要强调手部伸入画面
- 包含镜头晃动、失焦等真实感细节

### I) 视频编辑
角色替换、剧情颠覆、元素增减。

**提示词格式：**
```
基于@视频1，将[原元素]替换为[新元素]，其他部分保持不变。
```

**素材要求：**
- @视频1：需要编辑的原视频

**实际案例：角色替换**
```
基于@视频1，将原视频中的男性战士替换为女性机械师，保持所有动作、运镜和场景不变。

角色设定：女性机械师，短发，工装背心，机械手臂，护目镜挂在额头。

动作复刻：
- 原视频0-3秒的"挥剑动作" → 改为"挥动扳手"
- 原视频3-6秒的"格挡动作" → 改为"用机械臂格挡"
- 原视频6-9秒的"冲刺动作" → 保持相同的冲刺姿态和速度

场景保持：工业废墟背景、昏暗光线、火光照明，与原视频完全一致。

运镜保持：低角度仰拍 → 快速推进 → 环绕旋转，与原视频的运镜轨迹和速度完全一致。

电影级写实摄影，保持原视频的色调和质感。
```

**实际案例：元素增减**
```
基于@视频1，在原视频的战斗场景中增加雨天效果，其他保持不变。

增加元素：
- 大雨倾盆，雨滴在镜头前形成动态模糊
- 地面积水反射火光和人物剪影
- 角色衣物被雨水打湿，贴在身上
- 雨水顺着武器滴落

保持不变：
- 角色外观、动作、表情
- 运镜方式和轨迹
- 场景布局和光线方向

音效增加：雨声、水花溅起声、雷声。

电影级写实摄影，雨水具有真实的物理效果和光线折射。
```

**最佳实践：**
- 明确列出需要替换/增加/删除的元素
- 明确列出需要保持不变的元素
- 替换元素时保持动作的物理逻辑
- 增加元素时考虑与原场景的交互

```
0-2秒：镜头缓慢推进，女性舞者站在霓虹灯舞台中央，身体随节奏轻微摆动。
```

**@音频1要求：**

**最佳实践：**
- 精确计算每拍的时间间隔
- 在重拍位置安排视觉冲击力强的动作
- 使用镜头运动（推拉摇移）配合节奏
- 特效触发时机与节拍同步

### ≤15秒视频

## 内容类型专属策略

基于 1319 条真实 Seedance 2.0 提示词数据分析，不同内容类型有显著不同的最佳实践。详细模板见 `references/seedance-style-library.md`。

### 动作/打斗类

**结构：** 时间轴分镜式，每段 2-4 秒
**必须包含：**
- 低角度仰拍 + 慢镜头标注（升格）
- 物理冲击感描述（瓦片碎裂、衣服撕裂、火花四溅）
- 360度环绕镜头捕捉高速移动
- 结尾吊臂拉高成全景

**禁止：** 在一段中混写多种运镜（如"推进后环绕再拉远"）

### 舞蹈/MV 卡点类

**结构：** FORMAT 声明式（英文）或时间轴分镜式（中文）
**必须包含：**
- BPM 标注（如 128 BPM）
- 卡点时间精确到 0.5 秒
- 镜头跟随节拍的运动描述
- 多景别快切（特写→全景→中景）

**关键词：** 霓虹高饱和冷暖对比、浅景深突出动作、2.35:1宽银幕、24fps

### 婚礼/情感类

**结构：** 时间轴分镜式，每段 4-5 秒
**必须包含：**
- 丁达尔效应（耶稣光）或特定光效描述
- 角色外貌细节（面部特征精确描述）
- 末尾音效列表（乐器 + 自然音 + 环境音）

**禁止：** 文学化描述（如"爱意在空气中流淌"）→ 改为具体画面（如"新郎将下巴轻轻抵在新娘头顶"）

### 动物拟人/ASMR 类

**结构：** 步骤分镜式（按操作步骤切分）
**必须包含：**
- 动物职业/角色设定（如"专业拉面师傅"）
- 每个步骤的具体操作动作
- ASMR 音效列表（操作音 + 环境音 + 细节音）
- @图片1 锁定动物外观

### 恐怖/悬疑类

**结构：** 时间轴分镜式，节奏克制
**必须包含：**
- 环境底噪描述（荧光灯嗡嗡声、远处列车声）
- 恐怖元素的出现方式（渐进式，不要突然出现）
- 高潮处理：突然的静默比音效更有效
- 固定镜头 + 缓慢推进（比手持跟拍更恐怖）

## 音效描述规范

**真实数据发现：** 高质量提示词中，音效描述是被最严重低估的要素。在 1319 条提示词中，顶级案例几乎都在末尾单独列出音效层次。

### 音效描述位置

放在提示词末尾，禁止项声明之前：

```
[时间轴分镜内容]

音效：[音效列表]。
禁止出现文字、字幕、LOGO或水印。
```

### 音效层次结构

按以下三层描述，每层 1-3 个元素：

1. **音乐层**：配乐风格或乐器（如"空灵的吉他拨弦"、"电音节拍"、"爵士大提琴"）
2. **环境音层**：场景环境音（如"清脆的鸟鸣"、"远处车流声"、"荧光灯嗡嗡声"）
3. **动作音层**：具体动作产生的声音（如"踩在落叶上的沙沙声"、"金属碰撞声"、"甩面的破风声"）

### 各类型音效模板

| 内容类型 | 音效模板 |
|---------|---------|
| 动作/打斗 | 金属碰撞声、衣物破裂声、地面震动低频轰鸣、雨水拍打声、角色喘息声 |
| 舞蹈/MV | 舞步声、鞋底摩擦声、呼吸声、衣料摩擦声、饰品碰撞声（与节拍贴合） |
| 婚礼/情感 | 空灵吉他拨弦、清脆鸟鸣、踩落叶沙沙声、悠扬爱尔兰长笛 |
| 动物/ASMR | 操作工具声、沸腾水声、动作破风声、细节碰撞声 |
| 恐怖/悬疑 | 荧光灯嗡嗡声、远处列车声、脚步回声、突然的静默 |
| 科幻/赛博朋克 | 电子嗡鸣声、机械运转声、能量充能声、城市环境音 |

## 负向提示标准写法

基于真实数据中 76 条使用负向提示的案例，提炼出两套标准写法：

### 中文标准版（推荐，适合大多数场景）
```

```


### 英文标准版
```
No text, subtitles, logos, or watermarks. No cuts or transitions. No dialogue.
Avoid: blur, distortion, stiff movement, low quality, artifacts.
```

## 常见问题与解决方案

### Q1: 角色一致性不佳怎么办？
**解决方案：**
- 使用"一致性控制"模式（B模式）
- 提供高质量的参考图（清晰、正面、光线均匀）
- 在提示词中明确列出需要保持一致的特征
- 避免要求角色做过于复杂的动作

### Q2: 如何提升视频的真实感？
**解决方案：**
- 在提示词开头加上"电影级写实摄影，真实拍摄质感"
- 明确摄影机型号和镜头参数（如"ARRI Alexa Mini LF，Sigma 35mm f/1.4"）
- 描述具体的光源类型和方向
- 在结尾加上"避免3D渲染感、CG动画风格、卡通化效果"

### Q3: 视频时长超过15秒怎么处理？
**解决方案：**
- 使用分段拼接策略，将长视频拆分为多个15秒片段
- 每段独立完整，有明确的起承转合
- 在段与段之间设计清晰的转场（溶解、闪白、硬切）
- 标注拼接点和转场方式

### Q4: 如何让动作更流畅自然？
**解决方案：**
- 每个动作包含完整的动作链（准备→执行→恢复）
- 描述动作的物理细节（重心转移、肌肉紧绷、惯性）
- 避免"瞬移"式的突变，确保相邻时间段动作自然衔接
- 参考 `combat-choreography-guide.md` 中的物理逻辑要求

### Q5: 特效什么时候开始描述？
**解决方案：**
- 严格遵循手部描述五阶段模型
- 开头只描述自然的手部外观和魔法球
- 特效从捏爆魔法球的瞬间才开始出现
- 特效触发后，每一段都要描述特效的持续状态
- 详见 `prompt-writing-standards.md` 第1-150行

## 平台限制与注意事项

### 内容限制
- ❌ 不要上传含写实真人脸部的素材
- ❌ 避免暴力、血腥、恐怖内容
- ❌ 不要包含政治敏感内容
- ✅ 推荐使用虚构角色、动画风格、艺术化表现

### 技术限制
- 单次生成最长15秒
- 参考图分辨率建议 ≥1024x1024
- 参考视频分辨率建议 ≥720p
- @引用最多支持3个素材（@图片1、@视频1、@音频1）

### 最佳实践
- 提示词使用自然流畅的中文
- 避免过于复杂的场景和动作
- 优先使用平台推荐的能力模式
- 多生成几个版本，选择最佳效果
