# 战斗动作编排指南

本文档专门针对战斗类视频的动作编排，确保一镜到底的连贯性、物理逻辑性和Boss对战的完整性。

## 一、一镜到底战斗场景的核心原则

### 1.1 镜头连贯性

**一镜到底的本质：** 镜头从头到尾不切换，所有动作在同一个连续的镜头中完成。

**关键要求：**
- 镜头运动必须流畅，避免突然的方向改变
- 每个动作的结束位置是下一个动作的开始位置
- 镜头跟随主角的移动轨迹，保持主体在画面中的合理位置
- 使用自然的镜头过渡（推拉摇移跟升降）
- **场景转换必须标注物理遮挡物**（详见 1.3 节）

**第一人称POV一镜到底特点：**
- 镜头即主角的眼睛，所有运动模拟头部和身体的移动
- 奔跑时：上下轻微晃动（步伐节奏）
- 转身时：水平旋转，边缘动态模糊
- 受击时：剧烈晃动+短暂失焦
- 闪避时：快速侧移/后仰，产生倾斜和动态模糊
- 蓄力时：镜头稳定，轻微前推营造紧张感
- 释放大招时：镜头跟随能量方向前推，产生速度感

### 1.2 转场遮挡物标注（一镜到底必备）

**核心问题：** 一镜到底中如果需要场景切换（从室内到室外、从A地点到B地点），不写明转场方式，AI会直接硬切，破坏连贯性。

**解决方案：** 必须在需要场景切换的时间点，明确标注**物理遮挡物**作为视觉转场的连接点。

**什么是转场遮挡物：**
指在一镜到底中，利用镜头快速掠过的不透明物体，暂时遮挡全部画面，在遮挡的瞬间完成场景切换。观众感知为连续镜头，实际场景已经改变。

**常用转场遮挡物类型：**

| 遮挡物类型 | 适用场景 | 描述示例 |
|-----------|---------|---------|
| 建筑结构 | 室内外切换 | 镜头快速掠过石柱/墙壁/门框，掠过后场景变为… |
| 人物身体 | 角色间切换 | 角色转身时背部暂时充满画面，转过身后场景已变 |
| 自然遮挡 | 户外场景 | 镜头穿过茂密树丛/瀑布水帘，穿过后到达新场景 |
| 光效遮挡 | 特效场景 | 能量爆发产生强闪白/光晕充满画面，光效消散后场景切换 |
| 烟雾遮挡 | 战斗场景 | 爆炸产生的烟雾弥漫充满画面，烟雾散去后场景改变 |
| 快速横移 | 任何场景 | 镜头极速横移产生运动模糊，模糊消散后进入新场景 |

**提示词中的标注格式：**

```
转场遮挡：[遮挡物类型]（镜头快速掠过[具体遮挡物]，掠过后场景从[A]切换为[B]）
```

**❌ 错误：无遮挡物的场景切换**
```
5-6秒：主角在地下神殿中释放能量
6-7秒：主角站在空旷的演武场上
```
→ AI 会在两个完全不同的场景之间硬切，破坏一镜到底。

**✅ 正确：有遮挡物的无缝转场**
```
5-6秒：主角释放能量冲击波，巨大的闪白光效充满整个画面
转场遮挡：光效遮挡（能量闪白充满画面，闪白消散后场景从地下神殿切换为空旷的演武场）
6-7秒：光效消散，主角站在空旷的演武场上，远处可见山脉轮廓
```

**完整一镜到底转场示例：**
```
0-3秒：镜头跟随主角在古代宫殿内部向前行走，缓慢平移。
3-4秒：镜头掠过宫殿的巨大石柱，石柱暂时遮挡全部画面。
转场遮挡：建筑结构（镜头掠过石柱，石柱后方场景从古代宫殿切换为现代写字楼大厅）
4-6秒：镜头掠过石柱后，场景已变为现代写字楼大厅，主角继续向前行走。
```

**检查清单：**
- [ ] 一镜到底中是否需要场景切换？
- [ ] 每个场景切换点是否标注了物理遮挡物？
- [ ] 遮挡物类型是否与场景风格匹配？
- [ ] 遮挡前后的画面是否有合理的视觉衔接？

### 1.3 动作链完整性

**每个战斗动作必须包含三个阶段：**

1. **准备阶段（蓄力）** — 0.5-1秒
   - 身体重心调整
   - 肌肉紧绷，能量聚集
   - 视觉提示（光效开始汇聚、手部姿态变化）
   - 镜头配合：缓慢推近或稳定

2. **执行阶段（爆发）** — 0.3-0.5秒
   - 力量释放，动作执行
   - 特效触发，视觉高潮
   - 镜头配合：快速推进或震动

3. **恢复阶段（收招）** — 0.5-1秒
   - 动作完成，身体恢复平衡
   - 特效余波，环境反馈
   - 镜头配合：稳定或轻微后拉

**示例：释放火球术**
```
准备（1秒）：双手在胸前合拢，火焰能量在掌心汇聚，形成小型火球。
          镜头缓慢推近手部，焦点在火球上。

执行（0.5秒）：双手猛然向前推出，火球脱手飞出，拖着火焰尾迹。
            镜头跟随火球快速前推，产生速度感。

恢复（1秒）：双手保持推出姿态，手指微微颤抖（释放后的余震），
          身体重心恢复稳定。镜头回正，焦点回到主角。
```

## 二、Boss对战完整流程

### 2.1 标准Boss战结构

一个完整的Boss战应包含以下阶段：

**阶段1：小怪清场（3-5秒）**
- 小怪群的出现方式（从地面爬出/从侧面冲来/从空中降落）
- 小怪的攻击姿态（挥舞武器/张牙舞爪/能量蓄积）
- AOE技能释放，覆盖小怪群
- 小怪被击败的视觉反馈（身体僵直/倒地翻滚/能量消散）
- 场地清空，为Boss登场做准备

**阶段2：Boss登场（1-2秒）**
- Boss出现的方式（从天而降/从地面升起/传送门出现）
- Boss的威压感展示（体型、气场、特效）
- 镜头从俯视/平视切换到仰视，强调Boss的压迫感

**阶段3：Boss首次攻击（2-3秒）**
- Boss的攻击准备动作（蓄力、咆哮、能量汇聚）
- 攻击释放（光束/吐息/冲击波）
- 攻击轨迹清晰可见

**阶段4：主角闪避（1-2秒）**
- 观察到攻击来袭（镜头锁定攻击）
- 身体快速反应（侧移/后仰/翻滚）
- 攻击擦身而过或击中地面
- 镜头跟随闪避动作，产生动态感

**阶段5：主角反击准备（1-2秒）**
- 闪避完成，身体恢复平衡
- 开始蓄力大招（结印/能量汇聚/武器充能）
- 镜头稳定，焦点在蓄力动作上

**阶段6：终极大招释放（2-3秒）**
- 大招释放瞬间（光柱/能量波/巨型法术）
- 攻击轨迹清晰，速度感强烈
- 镜头跟随攻击方向快速前推

**阶段7：Boss受击与结束（1-2秒）**
- 大招命中Boss
- Boss的受击反应（咆哮/身体震动/能量爆炸）
- Boss被击败（崩解/消散/倒地）
- 镜头闪白或震动，强调冲击力

### 2.2 Boss对战时间分配示例（14秒视频）

```
0-3秒：小怪清场
  0-1秒：准备AOE技能
  1-2秒：释放AOE（环形闪电场扩散）
  2-3秒：小怪被击倒，场地清空

3-4秒：Boss登场
  3-3.5秒：烟雾散去，远处出现巨大阴影
  3.5-4秒：Boss从天空俯冲而下，双翼展开，镜头仰视

4-6秒：Boss首次攻击
  4-5秒：Boss口中蓄积能量球，发出咆哮
  5-6秒：Boss喷射能量吐息，粗壮光柱直冲主角

6-7秒：主角闪避
  6-6.5秒：主角身体向右侧急速闪避，镜头快速横移
  6.5-7秒：能量吐息擦身而过，击中身后地面，碎石飞溅

7-9秒：主角反击准备
  7-8秒：闪避完成，镜头回正，主角双手开始结印
  8-9秒：手部能量密度骤增，形成巨大法阵，符文闪烁

9-12秒：终极大招释放
  9-10秒：双手猛然推出，巨型能量光柱喷射而出
  10-11秒：光柱高速飞向Boss，沿途空气扭曲
  11-12秒：光柱命中Boss胸口，巨大爆炸

12-14秒：Boss被击败
  12-13秒：Boss被能量包裹，身体开始崩解
  13-14秒：Boss化为光点消散，手部保留能量余辉
```

## 三、物理逻辑细节清单

### 3.1 重力与惯性

**跳跃/飞行：**
- 起跳前：膝盖弯曲，身体下沉（蓄力）
- 上升时：身体向上加速，双臂辅助
- 最高点：速度为零，短暂悬停
- 下降时：身体向下加速，准备着地
- 落地时：膝盖弯曲缓冲，身体前倾

**奔跑/移动：**
- 启动时：身体前倾，后腿蹬地
- 奔跑中：身体保持前倾，手臂摆动
- 停止时：身体后仰（惯性），双脚减速
- 转向时：身体侧倾，重心转移

**闪避动作：**
- 侧闪：身体重心快速侧移，对侧腿蹬地，手臂摆动保持平衡
- 后仰：上身后倾，膝盖弯曲，双臂展开
- 翻滚：身体团缩，滚动一圈后单膝着地，手撑地面

### 3.2 力的传递

**拳击/近战攻击：**
```
力量路径：后腿蹬地 → 腰部旋转 → 肩部带动 → 手臂挥出 → 拳头/武器击中
```

**法术释放：**
```
能量路径：核心（丹田/胸口）→ 手臂 → 手掌 → 指尖 → 能量脱手
```

**大招释放的后坐力：**
- 能量释放瞬间，身体向后轻微震动
- 双脚向后滑动或在地面留下痕迹
- 手臂因反作用力微微颤抖
- 周围空气产生冲击波纹

### 3.3 环境交互

**地面反馈：**
- 重击地面：地面龟裂，碎石飞溅，尘土扬起
- 能量波扩散：推动地面尘土/水面/物体向外
- 脚步踩踏：水面溅起水花，沙地留下脚印

**空气扰动：**
- 快速移动：身后产生气流扰动，衣物飘动
- 能量释放：空气扭曲，产生热浪效果
- 爆炸冲击：冲击波向外扩散，推动周围物体

**光影变化：**
- 能量特效照亮周围环境
- 角色移动产生动态阴影
- Boss登场时天空变暗/变亮

## 四、第一人称POV战斗的特殊处理

### 4.1 手部动作描述

**手臂伸入画面的方式：**
- 施法时：双手从画面下方或两侧伸入，在画面中央汇聚
- 防御时：手臂快速抬起，遮挡部分视野
- 攻击时：手臂向前伸出，拳头/武器占据画面中心
- 结印时：双手在画面中央快速变换手势

**手部与镜头的距离感：**
- 近距离：手部占据画面1/3，细节清晰（结印、施法）
- 中距离：手部在画面边缘，展示动作轨迹（挥动、推出）
- 远距离：手部小幅度出现，焦点在目标上（指向、瞄准）

### 4.2 视线跟随

**目标锁定：**
- 小怪群：视线扫视，焦点在最近的敌人上
- Boss登场：视线从下往上移动（Boss从天而降）或从远到近（Boss冲来）
- Boss攻击：视线锁定攻击来源（Boss口中的能量球）
- 闪避时：视线始终追踪Boss位置，即使身体侧移也保持余光锁定

**焦点切换：**
- 从手部（施法）→ 目标（敌人）→ 特效（能量飞行）→ 结果（命中）
- 切换时使用自然的眼球运动（不是瞬移）

### 4.3 身体反馈

**受力反应：**
- 释放大招：身体向后轻微震动，镜头晃动
- 闪避攻击：身体快速侧移，镜头倾斜+横移
- 落地冲击：镜头向下震动，短暂失焦
- Boss咆哮：镜头轻微震动，模拟声波冲击

**呼吸与疲劳：**
- 战斗前：呼吸平稳，镜头轻微上下晃动
- 战斗中：呼吸加快，镜头晃动频率增加
- 大招后：呼吸急促，镜头晃动幅度增大

## 五、常见问题与解决方案

### 问题1：AOE清场后直接结束，缺少Boss战

**原因：** 时间线设计时只考虑了小怪战斗，忽略了Boss环节

**解决方案：**
1. 在时间线设计时，预留至少50%的时间给Boss战
2. 小怪清场控制在3-5秒内完成
3. Boss战至少需要6-8秒（登场→攻击→闪避→反击→结束）

**时间分配建议（15秒视频）：**
- 0-5秒：开场+小怪清场（33%）
- 5-13秒：Boss战（53%）
- 13-15秒：结束画面（14%）

### 问题2：Boss攻击后缺少主角反应

**原因：** 只描述了Boss的攻击动作，没有描述主角的应对

**解决方案：**
每次Boss攻击后，必须包含主角的反应：
- 闪避：身体移动，攻击落空
- 格挡：举起武器/护盾，承受冲击
- 受击：身体震动，后退几步
- 反击：趁Boss攻击间隙，立即反击

### 问题3：大招释放缺少蓄力过程

**原因：** 直接描述大招释放，缺少准备阶段

**解决方案：**
大招必须包含三个阶段：
1. 蓄力（1-2秒）：结印/能量汇聚/武器充能
2. 释放（0.5-1秒）：能量爆发，攻击发出
3. 命中（0.5-1秒）：攻击到达目标，产生效果

### 问题4：动作之间出现"瞬移"

**原因：** 相邻时间段的动作没有衔接

**解决方案：**
- 每个动作的结束姿态 = 下一个动作的起始姿态
- 使用过渡动作连接（重心转移、身体调整、小幅度移动）
- 检查清单：
  - [ ] 上一段结束时，角色的身体姿态是什么？
  - [ ] 下一段开始时，角色的身体姿态是什么？
  - [ ] 两者之间是否需要过渡动作？

### 问题5：镜头运动不连贯

**原因：** 镜头方向突然改变，缺少平滑过渡

**解决方案：**
- 使用箭头描述完整运镜路径：`推近 → 跟随 → 环绕 → 拉远`
- 避免突然的方向改变，使用弧线运动
- 关键动作时镜头配合：
  - 蓄力：缓慢推近
  - 爆发：快速推进或震动
  - 移动：跟随角色
  - 结束：稳定或轻微后拉

## 六、检查清单

生成战斗类视频提示词后，必须检查：

### 一镜到底连贯性
- [ ] 镜头从头到尾没有切换？
- [ ] 镜头运动路径清晰流畅？
- [ ] 每个动作的结束位置是下一个动作的开始位置？
- [ ] 镜头跟随主角移动，保持主体在画面中？

### Boss对战完整性
- [ ] 是否包含Boss登场环节？
- [ ] Boss是否有攻击动作？
- [ ] 主角是否有闪避/格挡反应？
- [ ] 是否有终极大招释放？
- [ ] Boss是否有受击和被击败的描述？
- [ ] Boss战时长是否占总时长的50%以上？

### 动作物理逻辑
- [ ] 每个动作是否有准备→执行→恢复三阶段？
- [ ] 跳跃/落地是否符合重力规律？
- [ ] 快速移动/停止是否考虑惯性？
- [ ] 力量传递路径是否正确？
- [ ] 大招释放是否有后坐力描述？

### 环境交互
- [ ] 地面是否有反馈（龟裂、尘土、水花）？
- [ ] 空气是否有扰动（冲击波、热浪）？
- [ ] 光影是否随动作变化？
- [ ] 特效是否照亮周围环境？

### 第一人称POV特殊检查
- [ ] 手部动作是否从画面边缘伸入？
- [ ] 视线跟随是否自然？
- [ ] 身体反馈是否体现（震动、倾斜）？
- [ ] 镜头晃动是否符合动作逻辑？

## 七、示例模板

### 完整Boss战提示词模板（14秒）

```markdown
## 时间线

### 0-1秒：静立展示
运镜：第一人称POV视角，镜头平稳，轻微呼吸感晃动；
画面描述：手掌向上摊开，托举着魔法球，背景是演武场，远处3-5只小怪群从地面裂缝中爬出，身形低矮扭曲，发出嘶吼声快速冲来；
人物动作：身体站立，重心稳定，手臂自然下垂；
手部状态：自然状态，皮肤细腻，指甲光泽；
```

## 八、总结

确保战斗类视频提示词的质量，关键在于：

1. **一镜到底的连贯性** — 镜头流畅，动作衔接自然
2. **Boss对战的完整性** — 登场→攻击→闪避→反击→结束，缺一不可
3. **动作的物理逻辑** — 准备→执行→恢复，符合重力、惯性、力的传递
4. **环境的真实反馈** — 地面、空气、光影随动作变化
5. **第一人称POV的沉浸感** — 手部、视线、身体反馈模拟真实体验

遵循本指南，可以生成高质量、连贯、符合物理逻辑的战斗类视频提示词。
