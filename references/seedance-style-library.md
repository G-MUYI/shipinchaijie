基于 1319 条真实 Seedance 2.0 提示词数据提炼，按内容类型分类，供生成提示词时参考。

## 一、结构框架速查

### 框架1：FORMAT 声明式（英文长提示词首选）

适用：英文提示词、多镜头切换、MV/动作场景

```
FORMAT: 15s / [镜头数] SHOTS / [连续/多切] / no dialogue
STYLE: [风格描述] / [色调] / [画质] / [画幅比例]

Shot 01 (0:00–X:XX): [画面描述]
Shot 02 (X:XX–X:XX): [画面描述]
...
```

### 框架2：时间轴分镜式（中文提示词首选）

适用：中文提示词、叙事性内容、婚礼/情感/动作

```
[总述：场景 + 风格 + 光影基调，一句话]

0-X秒：[运镜]，[画面描述20-40字]。
X-X秒：[运镜]，[画面描述20-40字]。
...

[摄影机参数]。禁止出现文字、字幕、LOGO或水印。
```

### 框架3：自然叙述式（短视频/简单场景）

适用：15秒以内、单一场景、动物/趣味内容

```
[主体描述] + [动作] + [场景] + [风格关键词]
```

### 动作/打斗类

**核心要素：** 时间轴分镜 + 慢动作标注 + 物理冲击感 + 升格描述

```
[时间段] 低角度仰拍，[主角]从[位置]跃下，慢镜捕捉[细节]，落地瞬间[环境反应]，快速推镜到中景。
[时间段] 快速跟拍[主角]与[对手]交手，[武器]激烈碰撞，火花四溅，360度环绕镜头捕捉高速移动，[液体/粒子]甩出弧线。
[时间段] 升格慢动作，[主角]起跳，[动作细节]，镜头仰拍捕捉腾空瞬间。
[时间段] 恢复正常速度，[结果描述]，[主角]轻松落地，[收尾动作]。

电影级光影，超现实粒子特效，物理真实，角色一致性，8K超清，无畸变。
禁止出现文字、字幕、LOGO或水印。
```

## 三、负向提示标准模板

### 中文标准版（推荐）
```
排除：模糊，低清，噪点，水印，文字，logo，扭曲，变形，五官崩坏，动作僵硬，画面抖动，比例失调。
```

### 中文完整版（高要求场景）
```
禁止：任何文字、字幕、LOGO或水印。画面全部片段都不要出现字幕。
排除：模糊，低清，噪点，扭曲，变形，五官崩坏，动作僵硬，画面抖动，比例失调，塑料质感，CG感。
```

### 英文标准版
```
No text, subtitles, logos, or watermarks. No cuts or transitions. No dialogue.
Avoid: blur, distortion, stiff movement, low quality, artifacts.
```
