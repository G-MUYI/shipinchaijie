# v2.2.1 提示词模板统一性修复 - 2026-03-29

## 修复概述

本次更新主要解决了项目中提示词模板的不一致问题，确保所有文档使用统一的标准和术语。

## 修复内容

### 1. 统一真实感描述词汇 ✅

**问题：** 标准文档禁止使用CG相关词汇，但实际输出文件中仍在使用"CG特效"、"CG建模"等表述。

**修复：**
- 在 `templates/prompt-template.md` 中明确列出禁用词清单和推荐词汇
- 在 `references/prompt-writing-standards.md` 中添加详细的词汇对照表

**禁用词汇：**
- ❌ CG动画
- ❌ 3D渲染
- ❌ CG特效
- ❌ CG建模
- ❌ 动画风格
- ❌ 卡通化
- ❌ 渲染感（单独使用时）

**推荐词汇：**
- ✅ 电影级写实摄影
- ✅ 真实拍摄质感
- ✅ 照片级写实
- ✅ 影视级实拍

### 2. 统一首帧提示词格式 ✅

**问题：** 首帧提示词的开头格式不统一，有的使用"电影级写实渲染"，有的使用"电影级写实摄影"。

**修复：**
- 在 `templates/prompt-template.md` 中添加明确的格式规范
- 在 `SKILL.md` 中更新示例，统一使用"电影级写实摄影，真实拍摄质感"开头
- 明确禁止使用"电影级写实渲染"（包含"渲染"一词）

**统一格式：**
```
开头固定：电影级写实摄影，真实拍摄质感。
结尾固定：避免3D渲染感、CG动画风格、卡通化效果。
```

### 3. 统一手部描述阶段命名 ✅

**问题：** 不同文档使用三阶段和五阶段两种模型，命名不统一。

**修复：**
- 在 `references/prompt-writing-standards.md` 中将三阶段扩展为五阶段模型
- 统一使用以下阶段命名：
  1. 自然阶段（特效触发前）
  2. 星座手势/结印阶段（准备触发）
  3. 捏爆魔法球阶段（触发瞬间）
  4. 特效爆发与蔓延阶段（特效开始出现）
  5. 特效持续状态阶段（特效完全激活后）

- 更新完整示例对比和检查清单

### 4. 整合战斗编排内容 ✅

**问题：** 战斗编排规则在多个文件中重复，维护时容易遗漏更新。

**修复：**
- 在 `templates/prompt-template.md` 中添加引用说明：
  ```
  详细规则请参考 `references/combat-choreography-guide.md`
  ```
- 在 `templates/gemini-prompt.txt` 中添加相同的引用说明
- 保留五阶段流程概要，详细规则指向专门文档

### 5. 统一禁止项格式 ✅

**问题：** 通用平台和Seedance平台的禁止项格式不同，缺乏明确说明。

**修复：**
- 在 `references/prompt-writing-standards.md` 中添加两种格式的使用场景说明

**通用平台格式（Sora/可灵/Runway）：**
```
禁止出现文字、字幕、LOGO或水印。
```

**Seedance 2.0 平台格式：**
```
禁止：
- 任何文字、字幕、LOGO或水印
- 画面全部片段都不要出现字幕
```

### 6. 补充基础版模板细节 ✅

**问题：** 基础版模板缺少高潮段落0.3秒细分规则和手部描述五阶段规则。

**修复：**
- 在 `templates/prompt-template-basic.md` 中添加详细的时间分配规则
- 添加完整的手部描述五阶段说明和示例
- 添加手部特效描述的6条重要规则

## 使用建议

1. **生成新提示词时**：严格遵循新的统一标准，特别注意禁用词汇清单
2. **首帧提示词**：必须使用"电影级写实摄影，真实拍摄质感"开头
3. **手部描述**：遵循五阶段渐进模型，不要在开头就描述特效
4. **战斗场景**：参考 `references/combat-choreography-guide.md` 获取详细规则
5. **平台适配**：根据目标平台选择正确的禁止项格式
