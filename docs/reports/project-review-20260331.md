**专业版（prompt-writing-standards.md）：**
1. 自然阶段 - 只描述手部外观和魔法球

4. 爆发阶段 - 描述特效从爆裂点向手部蔓延

### 4.2 关键规则传递一致性

#### 规则：手部描述五阶段

**Gemini 分析提示词中的要求：**
```
- 手部描述要区分多个阶段：
  - 自然状态：皮肤颜色、指甲颜色/形状、手部姿态、是否有饰品、
    手中是否握持/悬浮魔法球（约乒乓球大小）
  - 星座手势/结印：根据十二星座特性设计的专属手势或结印动作
  - 捏爆魔法球：手部猛然用力捏紧，魔法球在掌心碎裂爆开的瞬间
  - 特效爆发：特效从魔法球爆裂点向外扩散，蔓延至手部、手臂的过程
  - 特效持续状态：特效触发后持续伴随手部直到视频结尾
```

**生成模板中的要求：**
```

捏爆动作： {手部猛然用力捏紧魔法球，魔法球在掌心碎裂爆开的瞬间动作描述}
特效触发时间： {特效开始出现的时间点，通常在捏爆魔法球的瞬间}
特效蔓延过程： {特效如何从魔法球爆裂点延展到手部、手臂乃至全身}
特效持续性： {特效一旦触发，必须持续伴随手部直到视频结尾}
```
