## 改进背景

用户反馈的4个问题：
1. ✅ 激活特效之后没有AOE范围攻击小怪的段落（v2.0.0已解决）
2. ✅ 缺少与BOSS大战时的闪避互动动作（v2.0.0已解决）
3. ✅ 释放AOE技能和大招时缺少结印手势的详细描述（v2.0.0已解决）
4. ✅ 首帧绘图提示词和生成的每个主题提示词不要出现CG动画渲染等词汇，要保持真实的特效风格（v2.1.0本次解决）

## v2.1.0 核心改进

### 1. 首帧提示词格式标准化

**改进前：**
```
一位身穿深蓝色法师长袍的年轻男性，站在古老石质祭坛前...
风格：电影实拍质感，ARRI Alexa Mini LF拍摄...照片级写实渲染。
```

**改进后：**
```
电影级写实渲染，第一人称视角，一只白皙纤细的女性右手...
全局光照，次表面散射皮肤材质，体积光穿透空气微粒。
风格：电影级写实摄影，真实拍摄质感...照片级写实。
避免3D渲染感、CG动画风格、卡通化效果。
```

**关键变化：**
- ✅ 开头固定为"电影级写实摄影，真实拍摄质感"或"电影级写实渲染"
- ✅ 增加真实感技术描述（全局光照、次表面散射、体积光）
- ✅ 结尾明确声明避免的风格类型

### 2. 禁用词汇清单

明确禁止在首帧和视频提示词中使用以下词汇：
- ❌ CG动画
- ❌ 3D渲染
- ❌ 动画风格
- ❌ 卡通化
- ❌ 建模
- ❌ 特效渲染

### 3. 真实感技术词汇强化

**必须使用的真实感描述：**
- ✅ 电影级写实摄影
- ✅ 真实拍摄质感
- ✅ 照片级写实
- ✅ 次表面散射（SSS）
- ✅ 全局光照（GI）
- ✅ 体积光（Volumetric Lighting）
- ✅ HDR高动态范围
- ✅ 物理正确的光影

### 4. 参考提示词示例

用户提供的参考风格：
```
电影级写实渲染，第一人称视角，一只白皙纤细的女性右手，
淡粉色椭圆形指甲，手心向上轻轻托举一颗乒乓球大小的金色透明魔法球，
球体内部有天秤符号发光，背景虚化为荒凉破败的工业区废墟，阴沉天空，
全局光照，次表面散射皮肤材质
```

**采纳要点：**
- 第一人称视角的手部特写描述
- 详细的手部细节（皮肤、指甲）
- 魔法球的具体尺寸和材质
- 背景虚化处理
- 真实感光照技术

## 使用指南

### 生成首帧提示词时的检查清单

1. **开头检查**
   - [ ] 是否以"电影级写实摄影，真实拍摄质感"或"电影级写实渲染"开头？

2. **内容检查**
   - [ ] 是否包含详细的手部描述（皮肤、指甲、姿态）？
   - [ ] 是否包含魔法球的具体尺寸和材质？
   - [ ] 是否包含场景环境描述？
   - [ ] 是否包含光线描述（全局光照、次表面散射、体积光）？

3. **技术参数检查**
   - [ ] 是否包含摄影机型号（ARRI Alexa Mini LF）？
   - [ ] 是否包含镜头参数（Sigma 35mm f/1.4，f/2.8，ISO 1600）？

4. **禁用词汇检查**
   - [ ] 是否避免使用"CG动画"、"3D渲染"、"动画风格"、"卡通化"？

5. **结尾检查**
   - [ ] 是否包含"避免3D渲染感、CG动画风格、卡通化效果"声明？
