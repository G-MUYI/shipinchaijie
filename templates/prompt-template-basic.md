# 基础版提示词模板

适合新手用户的简化版本，保留核心字段，降低学习成本。

---

## 提示词格式

### 开头总述

```
[时长]，[画幅比例]，[整体风格]，[场景描述]，[主角描述]，[整体运镜方式]。
```

### 逐段描述

按 1-2 秒分段，每段包含：

```
[起始秒]-[结束秒]秒：

运镜：[镜头运动方式]
画面：[具体画面描述，50字以上]
动作：[人物动作]
情绪：[人物表情和情绪]
台词：[台词内容]（如有）
声音：[环境音、音效]
转场：[到下一段的过渡]
```

### 结尾固定

```
禁止出现文字、字幕、LOGO或水印。
```

---

## 填写示例（3秒短视频）

**开头总述：**

3秒，竖屏9:16，现代都市夜景，街道霓虹灯光，年轻女性黑色短发白T恤，手持摄影跟拍。

**逐段描述：**

0-1.5秒：

运镜：手持跟拍
画面：夜晚街道，女生站在路灯下低头看手掌。前景是侧脸和手部，背景是模糊车流
动作：右手摊开，掌心向上
情绪：眉头微皱，疑惑
台词：无
声音：城市环境音，车流声
转场：无

1.5-3秒：

运镜：快速推近手部特写
画面：手掌中心突然出现蓝色光点，扩散为电弧能量
动作：手指绷紧，手掌后仰
情绪：瞳孔放大，震惊
台词：无
声音：能量嗡鸣声，电弧声
转场：能量闪烁定格

禁止出现文字、字幕、LOGO或水印。

---

## 写作要点

1. 画面描述要具体 - 不写"美丽"，写"霓虹灯在湿润路面形成光斑"
2. 动作要精确 - 不写"走路"，写"右手举机自拍，左手拉人奔跑"
3. 情绪要具体 - 不写"开心"，写"抬头露出无奈又好笑的表情"
4. 每段 1-2 秒，关键动作可更细
5. 结尾必须加固定声明

---

## 常用运镜术语

- 推镜头 - 镜头向前推进
- 拉镜头 - 镜头向后拉远
- 跟拍 - 跟随主体移动
- 手持 - 轻微晃动的真实感
- 航拍 - 从空中俯视
- 特写 - 聚焦局部细节
