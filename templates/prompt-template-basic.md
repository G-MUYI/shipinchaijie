# 基础版提示词模板

适合新手用户的简化版本，保留核心字段，降低学习成本。

---

## 提示词格式

### 开头总述

```
[时长]，[画幅比例]，[整体风格]，[场景描述]，[主角描述]，[整体运镜方式]。
```

### 逐段描述

按 1-2 秒分段，每段包含：

```
[起始秒]-[结束秒]秒：

运镜：[镜头运动方式]
画面：[具体画面描述，50字以上]
动作：[人物动作]
情绪：[人物表情和情绪]
台词：[台词内容]（如有）
声音：[环境音、音效]
转场：[到下一段的过渡]
```

### 结尾固定

```
禁止出现文字、字幕、LOGO或水印。
```

## 填写示例（3秒短视频）

**开头总述：**

3秒，竖屏9:16，现代都市夜景，街道霓虹灯光，年轻女性黑色短发白T恤，手持摄影跟拍。

**逐段描述：**

0-1.5秒：

运镜：手持跟拍
画面：夜晚街道，女生站在路灯下低头看手掌。前景是侧脸和手部，背景是模糊车流
动作：右手摊开，掌心向上
情绪：眉头微皱，疑惑
台词：无
声音：城市环境音，车流声
转场：无

1.5-3秒：

运镜：快速推近手部特写
画面：手掌中心突然出现蓝色光点，扩散为电弧能量
动作：手指绷紧，手掌后仰
情绪：瞳孔放大，震惊
台词：无
声音：能量嗡鸣声，电弧声
转场：能量闪烁定格

禁止出现文字、字幕、LOGO或水印。

## 首帧AI绘图提示词格式

**重要：** 在生成视频提示词之前，必须先生成首帧绘图提示词用于垫图。

### 格式要求

**开头固定：** 电影级写实摄影，真实拍摄质感。

**结尾固定：** 避免3D渲染感、CG动画风格、卡通化效果。

**严禁使用：** CG动画、3D渲染、CG特效、CG建模、动画风格、卡通化、渲染感

**推荐使用：** 电影级写实摄影、真实拍摄质感、照片级写实、实拍质感、影视级实拍

### 首帧提示词应包含

1. 主体描述 — 角色/物体的详细外观
2. 环境描述 — 场景、背景、空间关系
3. 光线描述 — 光源方向、光线质感、色温
4. 视角描述 — 镜头角度（第一人称POV/第三人称）
5. 画面构图 — 主体在画面中的位置、景别
6. 风格关键词 — 摄影机型号和镜头参数

### 示例

```
电影级写实摄影，真实拍摄质感。一只白皙纤细的女性右手，淡粉色椭圆形指甲，手心向上轻轻托举一颗乒乓球大小的深蓝色透明魔法球，球体内部有神秘符文发光。背景虚化为昏暗的地下神殿，石柱上刻满发光的符文，微弱的蓝色光芒从地面裂缝中透出。

光线：主光源来自地面裂缝的蓝色冷光，从下往上照亮手部和魔法球。全局光照，次表面散射皮肤材质，体积光穿透空气微粒。

风格：ARRI Alexa Mini LF拍摄，Sigma 35mm f/1.4镜头，光圈f/2.8，ISO 1600，色彩分级偏冷调，高对比度，自然颗粒感纹理，照片级写实。避免3D渲染感、CG动画风格、卡通化效果。
```

## 写作要点

1. **画面内容要完整** - 必须描述画面中具体出现了什么（人物外观、场景元素、物体道具），不能只写动作。例如：不写"主角释放技能"，要写"黑发女性主角身穿紫色法袍，双手前伸释放紫色能量波，背景是古老石柱大厅"
2. 画面描述要具体 - 不写"美丽"，写"霓虹灯在湿润路面形成光斑"
3. 动作要精确 - 不写"走路"，写"右手举机自拍，左手拉人奔跑"
4. 情绪要具体 - 不写"开心"，写"抬头露出无奈又好笑的表情"
4. 每段时间分配：
   - 常规动作：1-2 秒一段
   - 关键动作：可精确到 0.5 秒
   - 高潮段落：可更细，0.3 秒一个关键帧
5. 结尾必须加固定声明
6. **手部特效描述规则（重要）**：
   - 开头只描述自然的手部外观（皮肤、指甲、姿态）
   - 如有魔法球，描述为"手中握持/悬浮一颗乒乓球大小的XX魔法球"
   - 特效必须从触发瞬间才开始出现（如捏爆魔法球）
   - 特效出现后，每一段都要描述特效的持续状态
   - 不要在开头就写"手上闪烁着光芒"等特效描述

## 手部描述五阶段（如涉及魔法/特效）

如果视频包含手部魔法或特效，必须按以下五个阶段渐进描述：

1. **自然阶段** - 只描述手部外观和魔法球
2. **手势阶段** - 描述特定的手势或结印动作
3. **触发阶段** - 描述捏爆魔法球的瞬间
4. **爆发阶段** - 描述特效从爆裂点向手部蔓延
5. **持续阶段** - 后续每段都描述特效的持续状态

**示例：**
```
0-1秒：手部（白皙皮肤，粉色指甲）托举着乒乓球大小的蓝色魔法球
1-2秒：手指收拢，呈现特定手势
2-2.5秒：手部发力，魔法球在掌心碎裂，白光闪烁
2.5-3秒：蓝色能量从爆裂点向手掌蔓延
3-4秒：手部被蓝色能量包裹，持续发光
4-5秒：手部的蓝色能量持续存在，随动作流动
```

## 常用运镜术语

- 推镜头 - 镜头向前推进
- 拉镜头 - 镜头向后拉远
- 跟拍 - 跟随主体移动
- 手持 - 轻微晃动的真实感
- 航拍 - 从空中俯视
- 特写 - 聚焦局部细节
