# 提示词输出格式骨架

生成提示词时，直接按以下骨架输出，不要改变结构层次。
根据用户选择的平台，使用【通用平台】或【Seedance 2.0】的骨架。

---

## 【通用平台骨架】Sora / 可灵 / Runway

### 输出结构（严格按此顺序）

```
# [主题名称]视频提示词

## 首帧AI绘图提示词

电影级写实摄影，真实拍摄质感。[主体：角色/手部详细描述]。[环境：场景、背景、氛围]。
光线：[光源方向、色温、光线质感，体积光穿透空气微粒]。
风格：[摄影机型号] [镜头参数]，[色彩分级]，照片级写实。避免3D渲染感、CG动画风格、卡通化效果。
```

### 时间分段规则

| 场景类型 | 分段粒度 |
|---------|---------|
| 常规动作 | 1-2秒/段 |
| 关键动作（特效触发、打击瞬间） | 0.5秒/段 |
| 高潮节点（爆炸、变身、大招释放） | 0.3秒/段 |

**最低段落数 = 视频总时长 ÷ 2**（例如15秒视频至少7-8段）

## 【Seedance 2.0骨架】即梦平台专属

> Seedance 核心限制：根据复杂度灵活控制字数（简单100-300字，标准300-600字，复杂600-1000字），每段只用一种运镜，禁止文学化描述，角色用@图片代替文字

### 内容类型快速选择

根据视频内容类型选择对应骨架，详细模板见 `references/seedance-style-library.md`：

| 内容类型 | 推荐骨架 | 关键特征 |
|---------|---------|---------|
| 动作/打斗 | 时间轴分镜式 | 低角度仰拍 + 慢镜头 + 360度环绕 |
| 舞蹈/MV卡点 | FORMAT声明式（英文）或时间轴式（中文） | BPM标注 + 卡点时间 + 多景别快切 |
| 婚礼/情感 | 时间轴分镜式 | 丁达尔效应 + 角色外貌细节 + 音效列表 |
| 动物拟人 | 步骤分镜式 | 职业设定 + 操作步骤 + ASMR音效 |
| 恐怖/悬疑 | 时间轴分镜式 | 固定镜头 + 渐进式恐怖 + 静默处理 |
| 科幻/赛博朋克 | FORMAT声明式 | 霓虹色调 + 镜头参数 + 特效物理感 |
| 简单/趣味 | 自然叙述式 | 直接描述，无需分段 |

### Seedance 自检

生成后对照以下检查，有问题就修改：

- [ ] 字数符合场景复杂度？（简单100-300，标准300-600，复杂600-1000）
- [ ] 每个时间段只用了一种运镜？
- [ ] 没有出现隐喻/诗意描述（如"像星辰般闪耀"）？
- [ ] 角色外观用 @图片 引用，而不是大量文字描述？
- [ ] 图片生成提示词用的是中文？
- [ ] 末尾有音效描述？（高质量提示词的共同特征）
- [ ] 内容类型匹配了对应骨架？（参考上方快速选择表）

## 多主题视频的输出规则

如果视频包含**多个独立主题**（不同角色/场景/故事线），必须为每个主题生成独立的完整文件：

1. 每个主题一个完整的骨架输出（包含首帧提示词 + 总述区 + 时间线区）
2. 文件命名：`video-breakdown-[主题名]-[时间戳].md`
3. 在总输出开头说明："本视频包含N个独立主题，已生成N个独立提示词文件"

## 快速参考：常用镜头术语

**景别**：大远景 · 远景 · 全景 · 中景 · 近景 · 特写 · 大特写

**运镜**：推镜头 · 拉镜头 · 摇镜头 · 跟拍 · 环绕拍摄 · 手持跟拍（带呼吸感） · 航拍俯视 · 希区柯克变焦 · FPV极限手持

**轨迹写法**：`中景特写 → 推近手部 → 拉出至全景`（用箭头连接）

**禁用词汇**：CG动画、3D渲染、CG特效、动画风格、卡通化、渲染感

**必用词汇**：电影级写实摄影、真实拍摄质感、照片级写实、次表面散射、体积光、全局光照
