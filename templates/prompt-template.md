# AI 视频生成提示词

## 输出格式说明

基于视频拆解分析结果，生成两个部分：
1. **详细拆解报告** — 极其细致的分析文档，用于理解和参考
2. **AI 视频生成提示词** — 可直接用于 AI 视频生成工具（Sora、可灵、Runway 等）的提示词

---

```markdown
5. **主角反击准备阶段**（1-2秒）：
```

### Timeline 分段规则

- **粒度极细**：以1-2秒为基本单位，关键动作精确到0.5秒
- **手部描述分阶段**：
  - 特效出现前：描述为自然的手部状态（如”一只手（女性，粉色指甲）”），手中握持/悬浮一颗乒乓球大小的魔法球
  - 星座手势阶段：描述符合星座特性的专属手势或结印动作（如白羊座的冲锋握拳、双子座的双手交叉、天蝎座的蝎尾指印等）
  - 捏爆魔法球：精确描述手部猛然用力捏紧，魔法球在掌心碎裂爆开的瞬间，包括手指收紧、爆裂光效、碎片飞溅
  - 特效开始时：精确描述特效从魔法球爆裂点向手部蔓延的过程
  - 特效激活后：描述完整的特效状态
  - **特效持续性（极重要）**：特效一旦被触发，必须在后续每一个时间段都持续存在并伴随手部，直到视频结尾。不能只在触发瞬间出现一次就消失。后续每段的手部状态描述中都必须包含”持续环绕的XX特效”等持续性描述
- **小怪冲击描述**：如涉及战斗场景，必须描述大量小怪从四面八方朝主角冲击而来，包括小怪数量、类型、冲击速度、包围态势
- **画面描述至少50字**：包含前景、中景、背景所有视觉层次
- **每段必须包含**：时间范围 + 画面环境 + 人物动作 + 人物情绪 + 声音/台词 + 转场

## 第二部分：AI 视频生成提示词格式

### 重要说明

**十二星座视频必须分别生成独立提示词**：
- 每个星座对应一个完整的独立提示词
- 每个星座提示词开头必须标注星座名称（如”# 白羊座专属魔法球”）
- 每个星座提示词必须包含首帧AI绘画提示词
- 不同星座之间用明显分隔符区分

### 标准格式

提示词分为**首帧绘画提示词**、**总述区**和**时间线区**三大部分。

#### 首帧AI绘画提示词（每个星座独立生成）

**格式规范（必须严格遵守）：**
1. **开头固定格式**：必须以"电影级写实摄影，真实拍摄质感。"开头
2. **禁止使用**："电影级写实渲染"、"CG"、"3D"等词汇
3. **结尾固定格式**：必须包含"避免3D渲染感、CG动画风格、卡通化效果。"

```markdown
# [星座名称]专属魔法球
```

### 写作要求

#### 总述区要求

1. **核心主题要精准** — 一句话概括视频的核心概念、风格定位和情绪基调（如”现实科幻特摄；废墟崩塌；暗能量汇聚；重甲暗骑士变形”）
2. **角色设定要完整** — 包含面部、发型、服装、体态、表情基调的完整描述。如有参考图标注”严格1:1复刻参考图”
3. **视觉风格要具体** — 必须包含：摄影机/镜头参考（如”IMAX电影质感，Panavision C系列35mm镜头，f/4光圈”）+ 主色调方案（如”低饱和度灰蓝色+暗金色”）+ 阴影处理 + 质感特征 + 视觉对比主题
4. **镜头运动用箭头轨迹** — 用箭头描述完整运镜路径（如”极限广角坠落 → 晃动推近至中景特写 → 变形环绕 → 拉远低角度双人构图”）

#### 时间线区要求

5. **段落主题标签** — 每段开头用简洁标签概括该段核心（如”凝视”、”激活”、”撕裂”、”完成”）
6. **画面描述要具象** — 不要写”美丽的场景”，要写”台北街头骑楼下，雨帘形成模糊光斑”
7. **动作要精确** — 不要写”她在走路”，要写”她一只手举机对着自己，另一只手拉着某人奔跑”
8. **情绪要具体** — 不要写”她很开心”，要写”抬头对镜头露出无奈又好笑的表情”
9. **光线与色调要逐段描述** — 每段标注该时刻的光线变化和色彩倾向（如”逆光剪影，暗金色能量照亮面部，阴影加深”）
10. **运镜要描述轨迹** — 不只写”手持跟拍”，要写”镜头跟随坠落、落地震动，快速推近面部特写”
11. **特效要分段细化** — 每段描述该时刻的特效细节（如”暗金属碎片向外迸发，装甲以吞噬方式组装”）
12. **台词带语气标注** — 台词（喘着气又笑）：”内容”
13. **无台词段落标注声音** — [雨声+街道环境音，无台词]
14. **每段时间精确** — 按1-2秒分段，关键节点可以更细
15. **结尾固定** — 以”禁止出现文字、字幕、LOGO或水印。”结束
16. **手部描述渐进式** — 开头总述只能写自然手部（含乒乓球大小的魔法球），特效从触发时刻才开始出现在描述中
17. **特效必须持续** — 特效一旦触发，后续每一段的手部状态都必须包含特效的持续描述（如”能量光芒持续环绕双手”、”魔法粒子持续缠绕指间”），不可中断或遗漏
18. **真实质感描述（极重要）** — 总述区和每段画面描述中必须强调真实拍摄质感。具体要求：
    - **严禁使用的词汇**：CG动画、3D渲染、CG特效、CG建模、动画风格、卡通化、渲染感
    - **推荐使用的词汇**：电影级写实摄影、真实拍摄质感、照片级写实、实拍质感、影视级实拍
    - 材质描述用”次表面散射的真实皮肤”、”物理正确的布料褶皱”、”真实金属菲涅尔反射”
    - 光照描述用”全局光照柔和弹射”、”体积光穿透空气微粒”、”特效光源真实照亮周围环境和面部”
    - 特效描述用”具有光照衰减的能量粒子”、”符合流体力学的烟雾扩散”、”真实空气扰动的冲击波”
    - 后期氛围用”电影级写实摄影”、”真实拍摄质感”、”照片级写实”、”自然大气散射的空间纵深”、”克制的Bloom溢出而非廉价发光”
    - 整体风格用”电影级写实摄影”、”真实拍摄质感”、”照片级写实”、”影视级实拍”
    - **首帧绘图提示词特别要求**：必须以”电影级写实摄影，真实拍摄质感”开头（不使用”渲染”一词），结尾必须包含”避免3D渲染感、CG动画风格、卡通化效果”
19. **战斗编排完整性** — 如涉及战斗场景，必须包含完整的：Boss攻击→主角闪避/防御→蓄力结印→大招释放的完整流程，不可跳过任何环节

## 第四部分：场景类型与提示词策略

根据视频内容类型，采用差异化的提示词侧重点：

### 电商/广告
- 产品360度旋转展示、爆炸分解、写实特效
- 第一人称沉浸式体验
- 配合广告词和品牌展示
- 侧重：材质细节、光影质感、产品特写

### 仙侠/动漫/特效
- 时间戳分镜法控制每段画面
- 详细的特效描述（法阵、能量波、粒子效果）
- 台词用引号标注，指定语气
- 侧重：画面真实感、特效物理感、战斗编排完整性

### 短剧/对白
- 画面与台词分开描述，台词标注角色和情绪
- 音效单独描述
- 精准时长控制
- 侧重：表演细节、情绪转换、对白节奏

### 科普教学
- 4K医学CGI风格或信息图形动画
- 半透明结构可视化
- 科学过渡自然流畅
- 侧重：信息清晰度、过渡逻辑、教育性旁白

### MV/音乐卡点
- 指定画幅比（2.35:1）和帧率（24fps）
- 分镜头描述每个镜头的场景、动作、音效
- 强调声音设计与节拍同步
- 侧重：节奏卡点、视觉冲击、画面与音乐的情绪匹配

## 第六部分：超长视频分段拼接策略

当视频超过单次生成上限时，采用分段生成+拼接的方式。

### 通用分段原则

1. 将总时长按叙事节奏切分为多个片段
2. 每段之间必须有**画面衔接点**：上一段的结尾状态 = 下一段的开始状态
3. 每段都要标注属于整体的第几段、承接内容是什么

### 分段时长建议

| 总时长 | 推荐分段 |
|--------|----------|
| 16-30秒 | 2段 |
| 31-45秒 | 3段 |
| 46-60秒 | 4段 |
| >60秒 | 拆分为独立场景分别生成，再用剪辑软件拼接 |

### 通用平台输出格式（Sora/可灵/Runway）

每段输出为独立的完整提示词，附衔接说明：

```markdown
## 超长视频提示词（总时长约Xs）

**主题**：[一句话概括]
**总段数**：[N段]
**建议比例**：[16:9 / 9:16 / 1:1]

---

### 第1段（0-15秒）

**生成时长**：15秒

#### 提示词

[完整提示词，时间戳分镜]

#### 衔接点

本段结尾画面：[精确描述结尾画面状态，用于下一段衔接]

---

### 第2段（15-30秒）

**承接上段**：[上段结尾画面状态]
**生成时长**：15秒

#### 提示词

[接续内容的完整提示词]

#### 衔接点

本段结尾画面：[精确描述结尾画面状态]
```

### Seedance 专属分段格式

Seedance 2.0 支持视频延长功能，后续段落使用 `将@视频1延长Xs` 格式：

```markdown
### 第2段（15-30秒）—— 视频延长

**操作**：将第1段生成的视频上传为 @视频1
**生成时长**：15秒

#### 提示词

将@视频1延长15秒。[接续内容的时间戳分镜描述]

#### 衔接点

本段结尾画面：[精确描述结尾画面状态]
```

#### @引用系统

Seedance 2.0 使用 `@` 语法引用上传的参考素材：

**官方命名规范：**
- 图片：`@图片1` ~ `@图片9`
- 视频：`@视频1` ~ `@视频3`
- 音频：`@音频1` ~ `@音频3`

**使用方式：** 在提示词中明确说明每个素材的用途：
- `@图片1为首帧` — 指定首帧画面
- `参考@视频1的运镜效果` — 借鉴运镜
- `背景音乐参考@音频1` — 音乐参考
- `@图片1的人物形象` — 角色一致性

**编号分配规则：**
1. 公共素材从 @图片1 开始依次编号
2. 版本独立素材（首帧、尾帧）在公共编号之后递增
3. 每个素材标题后标注对应 @编号

**注意事项：**
- 使用官方命名：`@图片1`（不是 @img1）、`@视频1`（不是 @video1）
- 写清楚是「参考」还是「编辑」
- **不支持上传含有写实真人脸部的素材**，系统会自动拦截

#### 多模态参考能力

- **首尾帧**：仅首帧图+提示词，适合简单场景
- **全能参考**：多模态组合输入（图片+视频+音频+文本），适合复杂创作

#### 十大能力提示词模式

##### 1. 纯文本生成（无参考素材）

**模式：** `(主体描述) + (动作序列) + (环境/光影) + (镜头语言) + (风格关键词)`

**示例：**
```
镜头跟随黑衣男子快速逃亡，后面一群人在追，镜头转为侧面跟拍，人物惊慌撞倒路边的水果摊爬起来继续逃，人群慌乱的声音。
```

##### 2. 一致性控制（角色/产品/场景统一）

通过参考图片保持人物、商品、场景的一致性。

**模式：** `[角色]@图片N + [动作/剧情] + [场景]@图片N + [运镜/光影]`

**示例：**
```
男人@图片1下班后疲惫的走在走廊，脚步变缓，最后停在家门口，脸部特写镜头，男人深呼吸，调整情绪，变得轻松，然后特写翻找出钥匙，插入门锁，进入家里后，小女儿和宠物狗欢快跑来迎接拥抱，室内温馨，全程自然对话
```

##### 3. 运镜与动作精准复刻

上传参考视频，复刻镜头语言、复杂动作、节奏变化。

**模式：** `参考@视频1的[运镜/动作/节奏] + [主体]@图片N + [场景描述]`

**示例：**
```
@图片1的女星作为主体，参考@视频1的运镜方式进行有节奏的推拉摇移，女星的动作也参考@视频1中女子的舞蹈动作，在舞台上活力十足地表演
```

##### 4. 创意模板/特效复刻

照着参考视频模仿创意转场、广告成片、电影片段。

**模式：** `参考@视频1的[特效/转场/创意] + 将[元素]替换为@图片N + [补充说明]`

**示例：**
```
黑白水墨风格，@图片1的人物参考@视频1的特效和动作，上演一段水墨太极功夫
```

##### 5. 剧情创作/补全

根据图片或分镜脚本自动生成剧情。

**模式：** `[分镜脚本/图片内容] + [演绎方式] + [音效/台词要求]`

**示例：**
```
参考@图片1的专题片分镜头脚本，参考其分镜、景别、运镜、画面和文案，创作一段15s的关于"童年的四季"的治愈系片头
```

##### 6. 视频延长

对已有视频进行平滑延长，可向前或向后。

**模式：**
```
将@视频1延长[X]s + [新增内容描述]
向前延长[X]s + [前置剧情描述]
```

**示例：**
```
将@视频1延长15秒。1-5秒：光影透过百叶窗在木桌、杯身上缓缓滑过，树枝伴随轻微晃动。6-10秒：一粒咖啡豆从画面上方轻轻飘落，镜头向咖啡豆推进至画面黑屏。11-15秒：英文渐显"Lucky Coffee"，"Breakfast"，"AM 7:00-10:00"。
```

> 注意：视频延长时，选择的生成时长应为"新增部分"的时长。

##### 7. 声音控制

支持音色参考、对白生成、音效设计。

**模式：** `[画面描述] + 音色/旁白参考@视频N + [台词内容用引号标注]`

**示例：**
```
根据提供的写字楼宣传照，生成一段15秒电影级写实风格的地产纪录片，采用2.35:1宽银幕，24fps，细腻的画面风格，其中旁白的音色参考@视频1
```

## 附录：专业术语词汇表

### 影视渲染术语

| 术语 | 解释 | 使用场景 |
|------|------|---------|
| 次表面散射（SSS） | 光线穿透半透明材质（如皮肤）后的散射效果 | 描述真实皮肤质感 |
| 全局光照（GI） | 光线在场景中多次反射形成的间接照明 | 强调真实光照环境 |
| 体积光 | 光线穿过烟雾、水汽等介质形成的可见光束 | 描述大气感、神秘氛围 |
| 菲涅尔反射 | 物体边缘反射率增强的物理现象 | 描述金属、玻璃等材质 |
| HDR | 高动态范围，保留亮部和暗部细节 | 强调画面层次丰富 |
| 物理正确的布料褶皱 | 符合重力和运动规律的布料形变 | 避免僵硬的数字感 |

### 光照术语

| 术语 | 解释 |
|------|------|
| 侧逆光 | 从侧后方照射的光线，形成轮廓光 |
| 丁达尔效应 | 光束穿过空气中的微粒形成可见光柱 |
| 漫射光 | 柔和均匀的散射光，无明显阴影 |
| 光照衰减 | 光线强度随距离递减的自然现象 |

### 后期术语

| 术语 | 解释 |
|------|------|
| Bloom | 高光溢出效果，模拟强光扩散 |
| 电影级调色 | 模拟胶片色彩的专业调色风格 |
| 自然大气散射 | 模拟空气对光线的散射，增强空间纵深 |
| 胶片颗粒感 | 模拟胶片的细微噪点，增加质感 |

使用这些术语时，确保与画面描述结合，避免堆砌关键词。
